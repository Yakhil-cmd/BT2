### Title
Staker Can Instantly Raise Commission to 100% via Commitment Mechanism, Stealing All Delegator Yield — (File: `src/staking/staking.cairo`)

---

### Summary
A staker (pool owner) can exploit the `set_commission_commitment` + `set_commission` flow to jump their commission from any low value to 100% (`COMMISSION_DENOMINATOR = 10000`) in a single epoch, redirecting all pool rewards to themselves and leaving delegators with zero yield for the duration of the exit wait window.

---

### Finding Description
The commission system has two paths in `update_commission`:

1. **No active commitment** → commission can only *decrease* (`commission < old_commission`).
2. **Active commitment** → commission can be *any value* ≤ `max_commission`, as long as it differs from `old_commission`. [1](#0-0) 

The `set_commission_commitment` function places no upper bound on `max_commission` other than `<= COMMISSION_DENOMINATOR` (10 000 = 100%). The only guard is that `max_commission >= current_commission`: [2](#0-1) 

This means a staker who currently charges 1% commission can:
1. Call `set_commission_commitment(max_commission: 10000, expiration_epoch: current_epoch + 1)` — valid because `100 <= 10000`.
2. Immediately call `set_commission(commission: 10000)` — valid because the commitment is active, `10000 <= 10000`, and `10000 != 100`.

The code itself acknowledges the gap: [3](#0-2) 

Once commission is 10 000, `split_rewards_with_commission` routes 100% of pool rewards to the staker and 0% to the pool: [4](#0-3) 

Delegators cannot exit instantly; they must wait through `exit_wait_window` (default 1 week, max 12 weeks): [5](#0-4) 

During that window every reward epoch distributes 0 STRK to the pool contract and all commission rewards to the staker's reward address.

---

### Impact Explanation
**High — Theft of unclaimed yield.**

All STRK rewards that would have accrued to delegators during the exit wait window are instead captured by the staker as commission. For a pool with significant TVL and a multi-week exit window, this can represent a material, permanent loss of yield for every pool member.

---

### Likelihood Explanation
**Medium-High.** The attack requires no special privilege beyond being a registered staker with an open pool. The two-transaction sequence (`set_commission_commitment` → `set_commission`) is atomic from the attacker's perspective and can be executed in consecutive blocks. A rational attacker would wait until the pool has accumulated substantial delegated stake before executing.

---

### Recommendation
1. **Cap `max_commission` in commitments** — e.g., require `max_commission <= current_commission + SOME_MAX_INCREASE` so a single commitment cannot jump to 100%.
2. **Enforce a time-delay on commission increases** — apply the new (higher) commission only after `K` epochs, giving delegators time to observe the change and exit before it takes effect.
3. **Alternatively**, treat the commitment as a strict ceiling that can never exceed the commission at the time delegators joined (i.e., require `max_commission <= commission_at_pool_open`).

---

### Proof of Concept

```
// Setup: staker has commission = 100 (1%), pool has delegators
// Step 1 – set commitment with max = 100% expiring next epoch
staking.set_commission_commitment(
    max_commission: 10000,          // COMMISSION_DENOMINATOR
    expiration_epoch: current_epoch + 1,
);

// Step 2 – immediately raise commission to 100%
staking.set_commission(commission: 10000);

// Result: split_rewards_with_commission(rewards, 10000)
//   commission_rewards = rewards * 10000 / 10000 = rewards  (all to staker)
//   pool_rewards       = rewards - rewards        = 0        (nothing to delegators)

// Delegators are trapped for exit_wait_window (up to 12 weeks) earning 0 yield.
``` [6](#0-5) [7](#0-6) [8](#0-7)

        fn set_commission(ref self: ContractState, commission: Commission) {
            // Prerequisites and asserts.
            self.general_prerequisites();
            assert!(commission <= COMMISSION_DENOMINATOR, "{}", Error::COMMISSION_OUT_OF_RANGE);
            let staker_address = get_caller_address();
            let staker_info = self.internal_staker_info(:staker_address);
            assert!(staker_info.unstake_time.is_none(), "{}", Error::UNSTAKE_IN_PROGRESS);

            let staker_pool_info = self.staker_pool_info.entry(staker_address);
            if let Option::Some(old_commission) = staker_pool_info.commission.read() {
                self
                    .update_commission(
                        :staker_address, :staker_pool_info, :old_commission, :commission,
                    );
            } else {
                staker_pool_info.commission.write(Option::Some(commission));
                self.emit(Events::CommissionInitialized { staker_address, commission });
            }
        }

        fn set_commission_commitment(
            ref self: ContractState, max_commission: Commission, expiration_epoch: Epoch,
        ) {
            self.general_prerequisites();
            assert!(max_commission <= COMMISSION_DENOMINATOR, "{}", Error::COMMISSION_OUT_OF_RANGE);
            let staker_address = get_caller_address();
            let staker_info = self.internal_staker_info(:staker_address);
            assert!(staker_info.unstake_time.is_none(), "{}", Error::UNSTAKE_IN_PROGRESS);
            let staker_pool_info = self.staker_pool_info.entry(staker_address);
            assert!(staker_pool_info.has_pool(), "{}", Error::MISSING_POOL_CONTRACT);
            let current_epoch = self.get_current_epoch();
            if let Option::Some(commission_commitment) = staker_pool_info
                .commission_commitment
                .read() {
                assert!(
                    !self.is_commission_commitment_active(:commission_commitment),
                    "{}",
                    Error::COMMISSION_COMMITMENT_EXISTS,
                );
            }
            // Staker must have a commission since it has a pool.
            let current_commission = staker_pool_info.commission();
            assert!(current_commission <= max_commission, "{}", Error::MAX_COMMISSION_TOO_LOW);
            assert!(expiration_epoch > current_epoch, "{}", Error::EXPIRATION_EPOCH_TOO_EARLY);
            assert!(
                expiration_epoch - current_epoch <= self.get_epoch_info().epochs_in_year(),
                "{}",
                Error::EXPIRATION_EPOCH_TOO_FAR,
            );
            let commission_commitment = CommissionCommitment { max_commission, expiration_epoch };
            staker_pool_info.commission_commitment.write(Option::Some(commission_commitment));

        fn update_commission(
            ref self: ContractState,
            staker_address: ContractAddress,
            staker_pool_info: StoragePath<Mutable<InternalStakerPoolInfoV2>>,
            old_commission: Commission,
            commission: Commission,
        ) {
            if let Option::Some(commission_commitment) = staker_pool_info
                .commission_commitment
                .read() {
                if self.is_commission_commitment_active(:commission_commitment) {
                    assert!(
                        commission <= commission_commitment.max_commission,
                        "{}",
                        Error::INVALID_COMMISSION_WITH_COMMITMENT,
                    );
                    assert!(commission != old_commission, "{}", Error::INVALID_SAME_COMMISSION);
                } else {
                    assert!(
                        commission < old_commission, "{}", Error::COMMISSION_COMMITMENT_EXPIRED,
                    );
                }
            } else {
                assert!(commission < old_commission, "{}", Error::INVALID_COMMISSION);
            }

            // Update commission in storage.
            staker_pool_info.commission.write(Option::Some(commission));

            // Emit event.
            self
                .emit(
                    Events::CommissionChanged {
                        staker_address, old_commission, new_commission: commission,
                    },
                );
        }

pub(crate) fn split_rewards_with_commission(
    rewards_including_commission: Amount, commission: Commission,
) -> (Amount, Amount) {
    let commission_rewards = compute_commission_amount_rounded_down(
        :rewards_including_commission, :commission,
    );
    let pool_rewards = rewards_including_commission - commission_rewards;
    (commission_rewards, pool_rewards)
}

pub(crate) fn compute_commission_amount_rounded_down(
    rewards_including_commission: Amount, commission: Commission,
) -> Amount {
    mul_wide_and_div(
        lhs: rewards_including_commission,
        rhs: commission.into(),
        div: COMMISSION_DENOMINATOR.into(),
    )
        .expect_with_err(err: InternalError::COMMISSION_ISNT_AMOUNT_TYPE)
}
