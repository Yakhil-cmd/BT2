### Title
Missing Temporal Declaration Check for Cairo 0 Classes in `ExecutionStateReader::get_compiled_class` (class_manager branch) — (`crates/apollo_rpc_execution/src/state_reader.rs`)

---

### Summary

When `class_manager_handle` is `Some`, `get_compiled_class` performs a temporal declaration check for Cairo 1 (V1) classes but unconditionally returns any Cairo 0 (V0) class the class_manager holds, regardless of whether that class was declared at or before `state_number`. This allows historical RPC simulation and fee estimation to execute using a Cairo 0 class that did not exist at the queried block.

---

### Finding Description

In `get_compiled_class`, the V1 branch (lines 122–134) calls `is_contract_class_declared` to verify the class was declared at or before `self.state_number` before returning it: [1](#0-0) 

The V0 branch (lines 136–140) carries an acknowledged TODO and skips this check entirely: [2](#0-1) 

The TODO explains the skip: `is_contract_class_declared` calls `get_class_definition_block_number`, which queries `declared_classes_block_table` — a table that only tracks Cairo 1 (Sierra) classes: [3](#0-2) [4](#0-3) 

For Cairo 0 classes, `get_class_definition_block_number` returns `None`, so `is_contract_class_declared` would always return `false` for them — rejecting all Cairo 0 classes, not just future ones. The correct check requires `get_deprecated_class_definition_block_number`: [5](#0-4) 

The class_manager does hold Cairo 0 classes via `add_deprecated_class`: [6](#0-5) 

The `class_manager_handle` is set to `Some` in production for `estimate_fee` and `simulate_transactions`: [7](#0-6) [8](#0-7) 

---

### Impact Explanation

An attacker who has previously deployed a proxy-style contract (e.g., one that reads a class hash from its own storage and issues a `library_call`) can:

1. Pre-compute the class hash of a Cairo 0 class before declaring it (class hashes are deterministic).
2. Deploy a contract at block N that stores this future class hash in its storage.
3. Declare the Cairo 0 class in block N+5; the class_manager now holds it.
4. Call `starknet_estimateFee` or `starknet_simulateTransactions` targeting block N with a transaction that invokes the proxy.
5. The proxy reads the stored class hash and issues a `library_call`; blockifier calls `get_compiled_class(X)`.
6. The class_manager returns `ContractClass::V0(X)`; the V0 branch returns it without any temporal check.
7. The simulation executes using class X — a class that did not exist at block N — producing wrong fee estimates, wrong execution traces, and wrong state diffs.

The `library_call` syscall takes a class hash directly from contract execution: [9](#0-8) 

The `StateReaderAndContractManager` (used in the gateway sequencing path) explicitly skips the `is_declared` check for V0 classes, mirroring the same assumption: [10](#0-9) 

---

### Likelihood Explanation

**Medium-low.** The attack requires the attacker to have deployed a suitable proxy contract on-chain before the historical query block, and to know the class hash of a Cairo 0 class before it is declared. Both conditions are achievable (class hashes are deterministic; proxy contracts are common), but require advance planning. The class_manager_handle branch is active in any production deployment that includes a class manager component.

---

### Recommendation

Replace the unconditional V0 return with a temporal check using `get_deprecated_class_definition_block_number`:

```rust
ContractClass::V0(deprecated_contract_class) => {
    let block_number = self
        .storage_reader
        .begin_ro_txn()
        .map_err(storage_err_to_state_err)?
        .get_state_reader()
        .map_err(storage_err_to_state_err)?
        .get_deprecated_class_definition_block_number(&class_hash)
        .map_err(|e| StateError::StateReadError(e.to_string()))?;

    match block_number {
        Some(bn) if self.state_number.is_after(bn) => {
            Ok(RunnableCompiledClass::V0(deprecated_contract_class.try_into()?))
        }
        _ => Err(StateError::UndeclaredClassHash(class_hash)),
    }
}
```

This mirrors the V1 check and resolves the acknowledged TODO.

### Proof of Concept

```rust
#[test]
fn get_compiled_class_v0_future_block_bypass() {
    use std::cell::Cell;
    use std::sync::Arc;
    use apollo_class_manager_types::MockClassManagerClient;
    use starknet_api::contract_class::ContractClass;
    use starknet_api::deprecated_contract_class::ContractClass as DeprecatedContractClass;
    use starknet_api::{class_hash};
    use starknet_api::block::{BlockHeader, BlockNumber};
    use starknet_api::state::{StateNumber, ThinStateDiff};
    use blockifier::state::state_api::StateReader;
    use blockifier::state::errors::StateError;
    use assert_matches::assert_matches;

    // class_hash X is a Cairo 0 class declared in block 1 (future relative to state_number=right_after(0))
    let class_hash_x = class_hash!("0xdeadbeef");
    let deprecated_class = DeprecatedContractClass::default();

    let mut mock = MockClassManagerClient::new();
    mock.expect_get_executable().returning(move |_| {
        Ok(Some(ContractClass::V0(deprecated_class.clone())))
    });

    let (mut exec_state_reader, mut storage_writer) =
        set_execution_state_reader(Arc::new(mock), 0); // state_number = right_after(block 0)

    // Write storage: class_hash_x declared in block 1 (future)
    storage_writer
        .begin_rw_txn().unwrap()
        .append_header(BlockNumber(0), &BlockHeader::default()).unwrap()
        .append_state_diff(BlockNumber(0), ThinStateDiff::default()).unwrap()
        .append_classes(BlockNumber(0), &[], &[]).unwrap()
        .append_header(BlockNumber(1), &BlockHeader::default()).unwrap()
        .append_state_diff(BlockNumber(1), ThinStateDiff {
            deprecated_declared_classes: vec![class_hash_x],
            ..Default::default()
        }).unwrap()
        .append_classes(BlockNumber(1), &[], &[(class_hash_x, &DeprecatedContractClass::default())]).unwrap()
        .commit().unwrap();

    // At state_number right_after(block 0), class_hash_x is NOT yet declared.
    // Expected: Err(UndeclaredClassHash(class_hash_x))
    // Actual (bug): Ok(V0 class) — temporal check is missing
    let result = exec_state_reader.get_compiled_class(class_hash_x);
    assert_matches!(
        result,
        Err(StateError::UndeclaredClassHash(h)) if h == class_hash_x,
        "BUG: V0 class from future block returned without temporal check"
    );
}
```

This test will **fail** (the assertion is not met) against the current code, demonstrating the missing check.
