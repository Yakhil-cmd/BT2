### Title

### Summary
`msgServer.TurnBridge` is implemented as a no-op stub. It performs no authorization check and writes no state. Any unprivileged address can submit `MsgTurnBridge` and receive a success response (code 0), making the `CanTurnBridge` permission bit permanently dead code and the bridge-enable/disable mechanism completely non-functional.

### Finding Description
`UpdateTokenMapping` correctly gates its logic behind a `HasPermission` call: [1](#0-0) 

`TurnBridge`, by contrast, is: [2](#0-1) 

The entire function body is `return nil, nil`. There is no call to `HasPermission`, no admin check, and no state mutation (no `SetParams` or equivalent). The `CanTurnBridge` permission constant is defined and tested in isolation: [3](#0-2) 

But it is never consulted inside `TurnBridge`. The `HasPermission` helper that would enforce it is fully functional and used elsewhere: [4](#0-3) 

### Impact Explanation
Two distinct failures exist simultaneously:

1. **Authorization bypass (High):** Any address â€” with no `CanTurnBridge` bit and regardless of whether `CronosAdmin` is set â€” can submit `MsgTurnBridge` and receive a success response. The permission system for this message is entirely unenforceable.

2. **Bridge state never changes:** Even a legitimately authorized caller cannot actually enable or disable the bridge via this message, because no state is written. The bridge-turn mechanism is completely broken.

This satisfies the **High** impact category: *Bypass of Cronos admin, governance authority, permission, token-mapping, bridge, block-list, or module-account authorization checks.*

### Likelihood Explanation
Trivially reachable. Any tx sender with a valid bech32 address can submit `MsgTurnBridge`. `ValidateBasic` only checks address format: [5](#0-4) 

No other guard exists between the tx and the stub handler.

### Recommendation
Replace the stub with the correct implementation, mirroring `UpdateTokenMapping`:

```go
func (k msgServer) TurnBridge(goCtx context.Context, msg *types.MsgTurnBridge) (*types.MsgTurnBridgeResponse, error) {
    ctx := sdk.UnwrapSDKContext(goCtx)

    senderAddr, err := sdk.AccAddressFromBech32(msg.Sender)
    if err != nil {
        return nil, errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid sender: %s", err)
    }
    if !k.HasPermission(ctx, []sdk.AccAddress{senderAddr}, CanTurnBridge) {
        return nil, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
    }

    params := k.GetParams(ctx)
    params.EnableAutoDeployment = msg.Enable  // or whichever field controls bridge state
    if err := k.SetParams(ctx, params); err != nil {
        return nil, err
    }
    return &types.MsgTurnBridgeResponse{}, nil
}
```

```go
func (k Keeper) HasPermission(ctx sdk.Context, accounts []sdk.AccAddress, permissionsToCheck uint64) bool {
	// case when no permission is needed
	if permissionsToCheck == 0 {
		return true
	}
	admin := k.GetParams(ctx).CronosAdmin
	for _, account := range accounts {
		if admin == account.String() {
			return true
		}
		permission := k.GetPermissions(ctx, account)
		if permission&permissionsToCheck == permissionsToCheck {
			return true
		}
	}

	return false
```
