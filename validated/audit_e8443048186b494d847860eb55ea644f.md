### Title
Source-Denom Token Mapping Is Permanently Irremovable, Blocking Emergency Response to a Compromised CRC21 Contract - (`x/cronos/keeper/keeper.go`)

### Summary

Once a `cronos0x...` source-denom token mapping is registered, neither the admin nor governance can remove or replace it. The `RegisterOrUpdateTokenMapping` handler has no deletion path for source coins, and `SetExternalContractForDenom` blocks re-registration via `ensureDenomNotMapped`. If the mapped CRC21 contract is later found to be vulnerable, the admin cannot sever the mapping to stop unauthorized minting of native `cronos0x...` tokens.

### Finding Description

`RegisterOrUpdateTokenMapping` in `x/cronos/keeper/keeper.go` branches on `types.IsSourceCoin(msg.Denom)`:

- **Non-source denoms** (`ibc/...`, `gravity0x...`): passing an empty `Contract` field triggers `DeleteExternalContractForDenom`, allowing removal.
- **Source denoms** (`cronos0x...`): the code unconditionally requires a valid, non-empty contract address and calls `SetExternalContractForDenom`. There is no empty-contract deletion path. [1](#0-0) 

`SetExternalContractForDenom` itself calls `ensureDenomNotMapped`, which returns `ErrDenomAlreadyMapped` if the denom already resolves to a contract: [2](#0-1) [3](#0-2) 

The governance path (`TokenMappingChangeProposal`) enforces the same constraint — source denoms always require a valid contract address, with no empty/delete option: [4](#0-3) 

The low-level `DeleteExternalContractForDenom` function exists and can delete source-denom mappings at the keeper level, but it is not reachable through any admin message or governance proposal for source coins: [5](#0-4) 

The spec explicitly acknowledges this: "There's no way to delete a token mapping currently." [6](#0-5) 

### Impact Explanation

When a source-denom CRC21 contract is compromised (e.g., contains a vulnerability allowing unauthorized minting of EVM tokens), the attack path is:

1. Attacker exploits the CRC21 contract to mint EVM tokens without authorization.
2. Attacker calls `ConvertCoinFromCRC21ToNative` (or emits `__CronosSendToIbc` to trigger the EVM hook).
3. The keeper calls `transfer_by_cronos_module` on the contract — this succeeds because the module address is the legitimate caller.
4. The keeper calls `bankKeeper.MintCoins` and sends native `cronos0x...` tokens to the attacker. [7](#0-6) 

The admin cannot stop this flow because the mapping is permanently locked. This constitutes **unauthorized minting of native tokens** (Critical) and **permanent bypass of admin authority over token mappings** (High).

### Likelihood Explanation

Any deployed `cronos0x...` source-denom mapping is permanently at risk if the underlying CRC21 contract is later found to be vulnerable. The admin has no emergency lever — no message, no governance proposal — to sever the mapping. The `DeleteExternalContractForDenom` keeper function exists but is unreachable from any on-chain transaction surface for source denoms.

### Recommendation

Expose a deletion path for source-denom mappings through `MsgUpdateTokenMapping` (or a dedicated `MsgDeleteTokenMapping`) restricted to the admin/governance authority. In `RegisterOrUpdateTokenMapping`, add a branch analogous to the non-source path:

```go
if types.IsSourceCoin(msg.Denom) {
    if len(msg.Contract) == 0 {
        k.DeleteExternalContractForDenom(ctx, msg.Denom)
        return nil
    }
    // ... existing registration logic
}
```

Similarly, update `TokenMappingChangeProposal.ValidateBasic` to permit an empty contract for source denoms (signalling deletion), and update the proposal handler to call `DeleteExternalContractForDenom` in that case. [4](#0-3) [8](#0-7) 

### Proof of Concept

1. Admin registers a source-denom mapping:
   ```
   MsgUpdateTokenMapping{Denom: "cronos0x<addr>", Contract: "<addr>", ...}
   ```
   This succeeds and stores the mapping.

2. Admin attempts to remove the mapping (emergency response):
   ```
   MsgUpdateTokenMapping{Denom: "cronos0x<addr>", Contract: "", ...}
   ```
   This fails at `ValidateBasic` — `common.IsHexAddress("")` is false, so the message is rejected before reaching the keeper. [9](#0-8) 

3. Admin attempts to remap to a safe contract:
   ```
   MsgUpdateTokenMapping{Denom: "cronos0x<addr>", Contract: "<addr>", ...}
   ```
   This fails at `SetExternalContractForDenom` → `ensureDenomNotMapped` → `ErrDenomAlreadyMapped`. [10](#0-9) 

4. Meanwhile, attacker exploits the vulnerable CRC21 contract, mints EVM tokens, and calls `ConvertCoinFromCRC21ToNative`. The keeper mints native `cronos0x...` tokens unconditionally because the mapping still resolves. [11](#0-10)

// SetExternalContractForDenom sets denom→external CRC21 mapping for an unmapped denom.
// Caller is responsible for source-denom specific validation before calling this method.
func (k Keeper) SetExternalContractForDenom(ctx sdk.Context, denom string, address common.Address) error {
	if err := k.ensureDenomNotMapped(ctx, denom); err != nil {
		return err
	}
	if err := k.ensureContractNotMapped(ctx, denom, address); err != nil {
		return err
	}

	store := ctx.KVStore(k.storeKey)
	store.Set(types.DenomToExternalContractKey(denom), address.Bytes())
	store.Set(types.ContractToDenomKey(address.Bytes()), []byte(denom))
	return nil
}

// DeleteExternalContractForDenom delete the external contract mapping for native denom,
// returns false if mapping not exists.
func (k Keeper) DeleteExternalContractForDenom(ctx sdk.Context, denom string) bool {
	store := ctx.KVStore(k.storeKey)
	contract, found := k.getExternalContractByDenom(ctx, denom)
	if !found {
		return false
	}
	store.Delete(types.DenomToExternalContractKey(denom))
	deleteReverseIfOwned(store, contract, denom)
	if auto, found := k.getAutoContractByDenom(ctx, denom); found {
		bz := store.Get(types.ContractToDenomKey(auto.Bytes()))
		if len(bz) == 0 {
			store.Set(types.ContractToDenomKey(auto.Bytes()), []byte(denom))
		} else if existingDenom := string(bz); existingDenom != denom {
			if k.contractOwnedByDenom(ctx, existingDenom, auto) {
				// auto address is already owned by another denom; drop local auto mapping
				store.Delete(types.DenomToAutoContractKey(denom))
			} else {
				// stale reverse entry
				store.Set(types.ContractToDenomKey(auto.Bytes()), []byte(denom))
			}
		}
	}
	return true
}

func (k Keeper) RegisterOrUpdateTokenMapping(ctx sdk.Context, msg *types.MsgUpdateTokenMapping) error {
	if types.IsSourceCoin(msg.Denom) {
		_, err := types.GetContractAddressFromDenom(msg.Denom)
		if err != nil {
			return err
		}

		if !common.IsHexAddress(msg.Contract) {
			return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
		}
		contract := common.HexToAddress(msg.Contract)
		if err := k.ensureContractCode(ctx, contract); err != nil {
			return err
		}
		if err := validateContractAddressForSourceDenom(msg.Denom, contract, true); err != nil {
			return err
		}

		if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
			return err
		}

		// check that the coin is registered, otherwise register it
		metadata, exist := k.bankKeeper.GetDenomMetaData(ctx, msg.Denom)
		if !exist {
			// create new metadata
			metadata = banktypes.Metadata{
				Base: msg.Denom,
				Name: msg.Denom,
			}
		}
		// update existing metadata
		metadata.Symbol = msg.Symbol
		metadata.Display = strings.ToLower(msg.Symbol)
		if msg.Decimal != 0 {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
				{
					Denom:    metadata.Display,
					Exponent: msg.Decimal,
				},
			}
		} else {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
			}
		}
		k.bankKeeper.SetDenomMetaData(ctx, metadata)
	} else {
		if len(msg.Contract) == 0 {
			// delete existing mapping
			k.DeleteExternalContractForDenom(ctx, msg.Denom)
		} else {
			if !common.IsHexAddress(msg.Contract) {
				return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
			}
			// update the mapping
			contract := common.HexToAddress(msg.Contract)
			if err := k.ensureContractCode(ctx, contract); err != nil {
				return err
			}
			if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
				return err
			}
		}
	}

func (k Keeper) ConvertCoinFromCRC21ToNative(ctx sdk.Context, contract, receiver common.Address, amount sdkmath.Int) error {
	denom, found := k.GetDenomByContract(ctx, contract)
	if !found {
		return fmt.Errorf("the contract address %s is not mapped to native token", contract.String())
	}

	isSource := types.IsSourceCoin(denom)
	coins := sdk.NewCoins(sdk.NewCoin(denom, amount))

	if isSource {
		_, err := k.CallModuleCRC21(ctx, contract, "transfer_by_cronos_module", receiver, amount.BigInt())
		if err != nil {
			return err
		}
		if err = k.bankKeeper.MintCoins(ctx, types.ModuleName, coins); err != nil {
			return err
		}
		if err = k.bankKeeper.SendCoinsFromModuleToAccount(ctx, types.ModuleName, sdk.AccAddress(receiver.Bytes()), coins); err != nil {
			return err
		}
