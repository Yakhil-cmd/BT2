### Title
Pool `claim_rewards` Bypasses Staking Contract Pause - (File: `src/pool/pool.cairo`)

### Summary
The pool contract's `claim_rewards` function does not check whether the staking contract is paused, allowing delegators to claim rewards while the protocol is paused. The staking contract's own `claim_rewards` is gated behind `general_prerequisites()` which enforces the pause, but the pool's equivalent entry point has no such guard.

### Finding Description
The staking contract stores a `is_paused` flag and enforces it via `assert_is_unpaused` inside `general_prerequisites()`. Every state-changing function in the staking contract calls `general_prerequisites()` first, including `claim_rewards`: [1](#0-0) 

The internal helper that enforces the pause: [2](#0-1) 

The pool contract's `claim_rewards`, however, contains no equivalent check. It reads the cumulative rewards trace, computes the owed amount, and immediately transfers STRK to the reward address: [3](#0-2) 

The pool contract has no `is_paused` storage field and no call to `staking.is_paused()` anywhere in its implementation: [4](#0-3) 

The pause test suite confirms every staking-contract function is tested for pause enforcement, but there is no corresponding test for `pool.claim_rewards` when the staking contract is paused: [5](#0-4) 

### Impact Explanation
When the security agent pauses the staking contract (e.g., in response to a reward-calculation bug or an accounting anomaly), the intent is to freeze all protocol operations until the incident is resolved. Stakers cannot call `staking.claim_rewards` — it reverts with `CONTRACT_IS_PAUSED`. Delegators, however, can still call `pool.claim_rewards` on any pool contract. Any STRK already credited to the pool (via prior `update_rewards_from_staking_contract` calls) can be drained to reward addresses before the security team can assess or reverse the situation. If the pause was triggered precisely because rewards were over-distributed, this constitutes theft of unclaimed yield — a **High** impact under the allowed scope. At minimum it undermines the protocol's incident-response capability, a **Medium** impact (griefing / damage to protocol).

### Likelihood Explanation
The entry point is fully unprivileged: any pool member or their registered reward address can call `pool.claim_rewards` at any time. No special role or leaked key is required. The only precondition is that the staking contract is paused, which is a normal operational state triggered by the security agent. Likelihood is **medium** — it requires the pause to be active, but once active the bypass is trivially exercisable by any delegator.

### Recommendation
Add a pause check at the top of `pool.claim_rewards` (and, for completeness, `pool.change_reward_address`) by querying the staking contract's pause state:

```cairo
fn claim_rewards(ref self: ContractState, pool_member: ContractAddress) -> Amount {
    // Add pause guard:
    let staking_dispatcher = IStakingDispatcher {
        contract_address: self.staking_pool_dispatcher.contract_address.read(),
    };
    assert!(!staking_dispatcher.is_paused(), "{}", Error::CONTRACT_IS_PAUSED);
    // ... rest of function
}
```

Alternatively, expose a shared internal helper in the pool that mirrors `assert_is_unpaused` from the staking contract.

### Proof of Concept
1. Security agent calls `staking.pause()` — `is_paused` is set to `true`.
2. Staker calls `staking.claim_rewards(staker_address)` → reverts: `"Contract is paused"`.
3. Delegator calls `pool.claim_rewards(pool_member)` on any deployed pool contract → **succeeds**, transferring accumulated STRK rewards to `reward_address` with no pause check enforced. [3](#0-2) [6](#0-5)

        fn claim_rewards(ref self: ContractState, pool_member: ContractAddress) -> Amount {
            // Asserts.
            let mut pool_member_info = self.internal_pool_member_info(:pool_member);
            let caller_address = get_caller_address();
            let reward_address = pool_member_info.reward_address;
            assert!(
                caller_address == pool_member || caller_address == reward_address,
                "{}",
                Error::POOL_CLAIM_REWARDS_FROM_UNAUTHORIZED_ADDRESS,
            );

            let until_checkpoint = self.get_current_checkpoint(:pool_member);

            // Calculate rewards and update entry_to_claim_from.
            let (mut rewards, updated_entry_to_claim_from) = self
                .calculate_rewards(
                    :pool_member,
                    from_checkpoint: pool_member_info.reward_checkpoint,
                    :until_checkpoint,
                    entry_to_claim_from: pool_member_info.entry_to_claim_from,
                );
            rewards += pool_member_info._unclaimed_rewards_from_v0;
            pool_member_info._unclaimed_rewards_from_v0 = Zero::zero();
            pool_member_info.entry_to_claim_from = updated_entry_to_claim_from;
            pool_member_info.reward_checkpoint = until_checkpoint;

            // Write the updated pool member info to storage.
            self.write_pool_member_info(:pool_member, :pool_member_info);

            // Transfer rewards to the pool member.
            let reward_token = IERC20Dispatcher { contract_address: STRK_TOKEN_ADDRESS };
            reward_token.checked_transfer(recipient: reward_address, amount: rewards.into());

            // Emit event.
            self
                .emit(
                    Events::PoolMemberRewardClaimed {
                        pool_member, reward_address, amount: rewards,
                    },
                );

            rewards
        }

#[test]
fn test_pause() {
    let mut cfg: StakingInitConfig = Default::default();
    general_contract_system_deployment(ref :cfg);
    let staking_contract = cfg.test_info.staking_contract;
    let staking_dispatcher = IStakingDispatcher { contract_address: staking_contract };
    let staking_pause_dispatcher = IStakingPauseDispatcher { contract_address: staking_contract };
    let is_paused = load_one_felt(
        target: staking_contract, storage_address: selector!("is_paused"),
    );
    assert!(is_paused == 0);
    assert!(!staking_dispatcher.is_paused());
    let mut spy = snforge_std::spy_events();
    // Pause with security agent.
    cheat_caller_address_once(
        contract_address: staking_contract, caller_address: cfg.test_info.security_agent,
    );
    staking_pause_dispatcher.pause();
    let is_paused = load_one_felt(
        target: staking_contract, storage_address: selector!("is_paused"),
    );
    assert!(is_paused != 0);
    assert!(staking_dispatcher.is_paused());
    // Unpause with security admin.
    cheat_caller_address_once(
        contract_address: staking_contract, caller_address: cfg.test_info.security_admin,
    );
    staking_pause_dispatcher.unpause();
    assert!(!staking_dispatcher.is_paused());
    // Validate Paused and Unpaused events.
    let events = spy.get_events().emitted_by(contract_address: staking_contract).events;
    assert_number_of_events(actual: events.len(), expected: 2, message: "pause");
    assert_paused_event(spied_event: events[0], account: cfg.test_info.security_agent);
    assert_unpaused_event(spied_event: events[1], account: cfg.test_info.security_admin);
}
