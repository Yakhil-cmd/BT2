### Title
Missing Contract Existence Check in `CallModuleCRC21` Enables Silent Accounting Corruption on Destructed External CRC21 Contracts - (File: x/cronos/keeper/evm.go)

### Summary

`CallModuleCRC21` invokes `CallEVM` → `ApplyMessage` against a mapped CRC21 contract address without verifying the contract still has code. The EVM returns success (empty return data, no revert) for calls to nonexistent or destructed contracts. `ensureContractCode` is only called once at registration time inside `RegisterOrUpdateTokenMapping`; there is no runtime existence guard. If an external contract is destructed after registration, every subsequent `CallModuleCRC21` call silently succeeds, corrupting the native↔CRC21 token accounting.

### Finding Description

`CallModuleCRC21` packs the ABI call data and dispatches it via `CallEVM`: [1](#0-0) 

`CallEVM` calls `k.evmKeeper.ApplyMessage`. Per EVM semantics, a call to an address with no code returns success with empty return data — `res.Failed()` is `false` and `err` is `nil`. The caller receives `([]byte{}, nil)` and proceeds as if the contract executed correctly.

The only contract-existence guard is `ensureContractCode`, called exclusively inside `RegisterOrUpdateTokenMapping` at mapping registration time: [2](#0-1) [3](#0-2) [4](#0-3) 

After registration, if the external contract is destructed (via `selfdestruct`), the mapping entry in the KV store remains intact. All subsequent calls to `CallModuleCRC21` against that address silently succeed.

**Affected flow — non-source IBC voucher conversion (`ConvertCoinFromNativeToCRC21`, non-source path):** [5](#0-4) 

1. `bankKeeper.SendCoins` moves the user's native IBC coins to the contract address (Cosmos-side balance transfer — irreversible at this point).
2. `CallModuleCRC21(…, "mint_by_cronos_module", …)` is called on the destructed contract.
3. EVM returns success with empty data; `res.Failed()` is `false`; no error is returned.
4. No CRC21 tokens are minted. The function returns `nil`.
5. The caller (`ConvertVouchersToEvmCoins` → `MsgConvertVouchers` / IBC middleware `OnRecvPacket`) considers the conversion successful.

The user's IBC vouchers are permanently trapped at the now-codeless contract address with no CRC21 tokens issued.

**Affected flow — source token conversion (`ConvertCoinFromCRC21ToNative`, source path):** [6](#0-5) 

1. `CallModuleCRC21(…, "transfer_by_cronos_module", …)` is called on the destructed contract — succeeds silently, no CRC21 tokens are moved.
2. `bankKeeper.MintCoins` mints native tokens unconditionally.
3. `SendCoinsFromModuleToAccount` delivers them to the receiver.

Native tokens are minted without any CRC21 tokens being burned or transferred — an unauthorized mint.

The IBC middleware path also reaches this via `ConvertVouchersToEvmCoins`: [7](#0-6) 

### Impact Explanation

**Non-source path:** IBC vouchers sent by honest users via `MsgConvertVouchers` or the IBC `OnRecvPacket` middleware are permanently locked at the dead contract address. No CRC21 tokens are issued. The conversion flow reports success, so the user has no indication of failure. This is a permanent, irreversible loss of bridged assets.

**Source path:** If the destructed contract previously emitted a `__CronosSendToAccount` event (e.g., in the same transaction as `selfdestruct`), the EVM hook processes the log post-transaction and calls `ConvertCoinFromCRC21ToNative`. The `transfer_by_cronos_module` call silently succeeds, and native tokens are minted without any CRC21 tokens being consumed — an unauthorized balance increase.

### Likelihood Explanation

An external CRC21 contract registered via `MsgUpdateTokenMapping` can be destructed by its owner at any time after registration. The Cronos admin registers the mapping but does not control the contract's lifecycle. Once destructed, every user who calls `MsgConvertVouchers` for that denom, or every IBC packet that auto-converts via the middleware, silently loses funds. No privileged Cronos action is required to trigger the loss — any unprivileged user submitting a standard conversion transaction after the contract is destructed is affected.

### Recommendation

Add a contract existence check inside `CallModuleCRC21` (or as a guard in `ConvertCoinFromNativeToCRC21` / `ConvertCoinFromCRC21ToNative`) before dispatching the EVM call:

```go
func (k Keeper) CallModuleCRC21(ctx sdk.Context, contract common.Address, method string, args ...interface{}) ([]byte, error) {
    if err := k.ensureContractCode(ctx, contract); err != nil {
        return nil, fmt.Errorf("contract no longer exists at %s: %w", contract.Hex(), err)
    }
    // ... existing logic
}
```

This mirrors the guard already present at registration time and prevents silent success when the target contract has been destructed.

### Proof of Concept

1. Admin calls `MsgUpdateTokenMapping` registering external contract `C` for IBC denom `D`. `ensureContractCode` passes — `C` has code.
2. Owner of `C` calls `selfdestruct` on `C`. EVM clears `C`'s code. The Cronos KV store mapping `D → C` remains unchanged.
3. User calls `MsgConvertVouchers{address: user, coins: [{denom: D, amount: 100}]}`.
4. `ConvertVouchersToEvmCoins` → `ConvertCoinFromNativeToCRC21`:
   - `bankKeeper.SendCoins(user → C, 100D)` succeeds — 100 IBC vouchers now at address `C`.
   - `CallModuleCRC21(C, "mint_by_cronos_module", user, 100)` → `ApplyMessage` → EVM call to codeless address → returns `(success, empty_data)`.
   - `res.Failed()` is `false`; function returns `nil`.
5. Transaction succeeds. User's 100 IBC vouchers are permanently locked at `C`. User receives 0 CRC21 tokens. [1](#0-0) [8](#0-7) [9](#0-8)

func (k Keeper) CallModuleCRC21(ctx sdk.Context, contract common.Address, method string, args ...interface{}) ([]byte, error) {
	data, err := types.ModuleCRC21Contract.ABI.Pack(method, args...)
	if err != nil {
		return nil, err
	}
	_, res, err := k.CallEVM(ctx, &contract, data, big.NewInt(0), DefaultGasCap)
	if err != nil {
		return nil, err
	}
	if res.Failed() {
		return nil, fmt.Errorf("call contract failed: %s, %s, %s", contract.Hex(), method, res.Ret)
	}
	return res.Ret, nil
}

// ConvertCoinFromNativeToCRC21 convert native token to erc20 token
func (k Keeper) ConvertCoinFromNativeToCRC21(ctx sdk.Context, sender common.Address, coin sdk.Coin, autoDeploy bool) error {
	if !types.IsValidCoinDenom(coin.Denom) {
		return fmt.Errorf("coin %s is not supported for conversion", coin.Denom)
	}
	var err error
	// external contract is returned in preference to auto-deployed ones
	contract, found := k.GetContractByDenom(ctx, coin.Denom)
	if !found {
		if !autoDeploy {
			return fmt.Errorf("no contract found for the denom %s", coin.Denom)
		}
		contract, err = k.DeployModuleCRC21(ctx, coin.Denom)
		if err != nil {
			return err
		}
		if err = k.SetAutoContractForDenom(ctx, coin.Denom, contract); err != nil {
			return err
		}

		k.Logger(ctx).Info(fmt.Sprintf("contract address %s created for coin denom %s", contract.String(), coin.Denom))
	}

	isSource := types.IsSourceCoin(coin.Denom)
	coins := sdk.NewCoins(coin)
	if isSource {
		// burn coins
		err = k.bankKeeper.SendCoinsFromAccountToModule(ctx, sdk.AccAddress(sender.Bytes()), types.ModuleName, sdk.NewCoins(coin))
		if err != nil {
			return err
		}
		err = k.bankKeeper.BurnCoins(ctx, types.ModuleName, coins)
		if err != nil {
			return err
		}
		// unlock crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "transfer_from_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	} else {
		// send coins to contract address
		err = k.bankKeeper.SendCoins(ctx, sdk.AccAddress(sender.Bytes()), sdk.AccAddress(contract.Bytes()), coins)
		if err != nil {
			return err
		}
		// mint crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "mint_by_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	}

	return nil
}

func (k Keeper) HasContractCode(ctx sdk.Context, contract common.Address) bool {
	return k.ensureContractCode(ctx, contract) == nil
}

func (k Keeper) ensureContractCode(ctx sdk.Context, contract common.Address) error {
	if contract.Big().Cmp(big.NewInt(256)) < 0 {
		return errors.Wrapf(sdkerrors.ErrInvalidAddress,
			"crc21 contract must not be in precompile range: %s", contract.Hex())
	}
	resp, err := k.evmKeeper.Code(ctx, &evmtypes.QueryCodeRequest{
		Address: contract.Hex(),
	})
	if err != nil {
		return errors.Wrapf(sdkerrors.ErrInvalidAddress, "failed to query contract code (%s): %v", contract.Hex(), err)
	}
	if resp == nil || len(resp.Code) == 0 {
		return errors.Wrapf(sdkerrors.ErrInvalidRequest, "no contract code at address (%s)", contract.Hex())
	}
	return nil
}
