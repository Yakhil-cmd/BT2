### Title
Stale Delegated Permissions Persist After `CronosAdmin` Rotation, Allowing Formerly-Authorized Addresses to Corrupt Token Mappings - (File: x/cronos/keeper/permissions.go, x/cronos/keeper/params.go)

### Summary
When governance rotates the `CronosAdmin` parameter via `MsgUpdateParams`, all per-address permission entries previously written to the `AdminToPermissionsKey` KVStore by the old admin remain intact. `HasPermission` reads those entries directly without any validity check against the current admin, so any address that was granted `CanChangeTokenMapping` by the old admin retains that permission indefinitely and can continue to call `MsgUpdateTokenMapping` after the admin rotation.

### Finding Description
The Cronos module implements a two-layer permission system:

1. **`CronosAdmin`** — a bech32 address stored in `Params` (updated via governance `MsgUpdateParams`).
2. **Delegated permissions** — per-address `uint64` bitmasks stored under `AdminToPermissionsKey(address)` in the KVStore, written by the current admin via `MsgUpdatePermissions`.

`HasPermission` checks both layers:

```go
// x/cronos/keeper/permissions.go:32-48
func (k Keeper) HasPermission(ctx sdk.Context, accounts []sdk.AccAddress, permissionsToCheck uint64) bool {
    admin := k.GetParams(ctx).CronosAdmin
    for _, account := range accounts {
        if admin == account.String() {
            return true
        }
        permission := k.GetPermissions(ctx, account)   // reads KVStore directly
        if permission&permissionsToCheck == permissionsToCheck {
            return true
        }
    }
    return false
}
```

`SetParams`, called by `UpdateParams` when governance rotates the admin, only overwrites the `ParamsKey` entry and never touches the `AdminToPermissionsKey` namespace:

```go
// x/cronos/keeper/params.go:27-37
func (k Keeper) SetParams(ctx sdk.Context, params types.Params) error {
    if err := params.Validate(); err != nil {
        return err
    }
    store := ctx.KVStore(k.storeKey)
    bz := k.cdc.MustMarshal(&params)
    store.Set(types.ParamsKey, bz)
    return nil
}
```

There is no iterator over `KeyPrefixAdminToPermissions` exposed in the keeper, and `ExportGenesis` does not export permission entries, so the new admin has no reliable way to enumerate and revoke stale grants. [1](#0-0) [2](#0-1) [3](#0-2) 

### Impact Explanation
An address `addrX` that was granted `CanChangeTokenMapping` (bit 1) by the old admin retains that permission after the admin rotation. It can call `MsgUpdateTokenMapping`, which is gated solely by `HasPermission(..., CanChangeTokenMapping)`:

```go
// x/cronos/keeper/msg_server.go:69-81
func (k msgServer) UpdateTokenMapping(...) {
    if !k.HasPermission(ctx, msg.GetSigners(), CanChangeTokenMapping) {
        return nil, errors.Wrap(sdkerrors.ErrUnauthorized, ...)
    }
    if err := k.RegisterOrUpdateTokenMapping(ctx, msg); err != nil { ... }
}
```

`RegisterOrUpdateTokenMapping` can:
- **Delete** an existing denom→contract mapping (passing empty `Contract`), breaking IBC voucher conversion for that token and stranding user funds.
- **Register** a new mapping for any unmapped denom, pointing it at an attacker-controlled contract, corrupting the denom/contract binding used by `ConvertVouchersToEvmCoins` and EVM hooks.

This is a **High** impact: bypass of Cronos admin/governance authorization checks and corruption of token mappings with direct security impact on asset conversion flows. [4](#0-3) [5](#0-4) 

### Likelihood Explanation
Admin rotation via governance is an expected operational event (documented in ADR-009 and the params spec). Any address that was ever granted delegated permissions by a previous admin automatically becomes a stale-privileged actor after rotation. The new admin cannot enumerate stale grants (no `GetAllPermissions` iterator is exposed; `ExportGenesis` omits the `AdminToPermissionsKey` prefix), making silent cleanup impossible. The exploit requires only submitting a standard `MsgUpdateTokenMapping` transaction — no special tooling or privileged access. [6](#0-5) [7](#0-6) 

### Recommendation
In `SetParams`, when `CronosAdmin` changes, iterate over all `KeyPrefixAdminToPermissions` entries and delete them:

```go
func (k Keeper) SetParams(ctx sdk.Context, params types.Params) error {
    if err := params.Validate(); err != nil {
        return err
    }
    // If admin is rotating, purge all stale delegated permissions.
    old := k.GetParams(ctx)
    if old.CronosAdmin != params.CronosAdmin {
        store := ctx.KVStore(k.storeKey)
        iter := prefix.NewStore(store, types.KeyPrefixAdminToPermissions).Iterator(nil, nil)
        var keys [][]byte
        for ; iter.Valid(); iter.Next() {
            keys = append(keys, iter.Key())
        }
        iter.Close()
        for _, k := range keys {
            store.Delete(append(types.KeyPrefixAdminToPermissions, k...))
        }
    }
    store := ctx.KVStore(k.storeKey)
    bz := k.cdc.MustMarshal(&params)
    store.Set(types.ParamsKey, bz)
    return nil
}
```

Alternatively, expose a `GetAllPermissions` iterator and include permission entries in `ExportGenesis`/`InitGenesis` so the new admin can audit and revoke stale grants. [2](#0-1) [8](#0-7) 

### Proof of Concept

1. Governance sets `CronosAdmin = adminA` via `MsgUpdateParams`.
2. `adminA` sends `MsgUpdatePermissions{From: adminA, Address: addrX, Permissions: 1}` — grants `addrX` `CanChangeTokenMapping`. KVStore now has `AdminToPermissionsKey(addrX) = 0x0000000000000001`.
3. Governance passes a new `MsgUpdateParams` setting `CronosAdmin = adminB`. `SetParams` writes the new `ParamsKey` but leaves `AdminToPermissionsKey(addrX)` untouched.
4. `addrX` submits `MsgUpdateTokenMapping{Sender: addrX, Denom: "ibc/...", Contract: ""}`. `HasPermission` reads `GetPermissions(addrX) = 1`, passes the check, and `DeleteExternalContractForDenom` removes the IBC voucher→CRC20 mapping.
5. All subsequent `ConvertVouchers` calls for that IBC denom fail; user funds are stranded as native IBC vouchers with no EVM representation. [9](#0-8) [10](#0-9)

// DeleteExternalContractForDenom delete the external contract mapping for native denom,
// returns false if mapping not exists.
func (k Keeper) DeleteExternalContractForDenom(ctx sdk.Context, denom string) bool {
	store := ctx.KVStore(k.storeKey)
	contract, found := k.getExternalContractByDenom(ctx, denom)
	if !found {
		return false
	}
	store.Delete(types.DenomToExternalContractKey(denom))
	deleteReverseIfOwned(store, contract, denom)
	if auto, found := k.getAutoContractByDenom(ctx, denom); found {
		bz := store.Get(types.ContractToDenomKey(auto.Bytes()))
		if len(bz) == 0 {
			store.Set(types.ContractToDenomKey(auto.Bytes()), []byte(denom))
		} else if existingDenom := string(bz); existingDenom != denom {
			if k.contractOwnedByDenom(ctx, existingDenom, auto) {
				// auto address is already owned by another denom; drop local auto mapping
				store.Delete(types.DenomToAutoContractKey(denom))
			} else {
				// stale reverse entry
				store.Set(types.ContractToDenomKey(auto.Bytes()), []byte(denom))
			}
		}
	}
	return true
}

func (k Keeper) RegisterOrUpdateTokenMapping(ctx sdk.Context, msg *types.MsgUpdateTokenMapping) error {
	if types.IsSourceCoin(msg.Denom) {
		_, err := types.GetContractAddressFromDenom(msg.Denom)
		if err != nil {
			return err
		}

		if !common.IsHexAddress(msg.Contract) {
			return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
		}
		contract := common.HexToAddress(msg.Contract)
		if err := k.ensureContractCode(ctx, contract); err != nil {
			return err
		}
		if err := validateContractAddressForSourceDenom(msg.Denom, contract, true); err != nil {
			return err
		}

		if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
			return err
		}

		// check that the coin is registered, otherwise register it
		metadata, exist := k.bankKeeper.GetDenomMetaData(ctx, msg.Denom)
		if !exist {
			// create new metadata
			metadata = banktypes.Metadata{
				Base: msg.Denom,
				Name: msg.Denom,
			}
		}
		// update existing metadata
		metadata.Symbol = msg.Symbol
		metadata.Display = strings.ToLower(msg.Symbol)
		if msg.Decimal != 0 {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
				{
					Denom:    metadata.Display,
					Exponent: msg.Decimal,
				},
			}
		} else {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
			}
		}
		k.bankKeeper.SetDenomMetaData(ctx, metadata)
	} else {
		if len(msg.Contract) == 0 {
			// delete existing mapping
			k.DeleteExternalContractForDenom(ctx, msg.Denom)
		} else {
			if !common.IsHexAddress(msg.Contract) {
				return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
			}
			// update the mapping
			contract := common.HexToAddress(msg.Contract)
			if err := k.ensureContractCode(ctx, contract); err != nil {
				return err
			}
			if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
				return err
			}
		}
	}

	return nil
}
