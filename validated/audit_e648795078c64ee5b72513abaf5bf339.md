### Title
Missing Contract Existence Check Before EVM Call Enables Silent Conversion Failure and Asset Loss — (`x/cronos/keeper/evm.go`)

---

### Summary

`CallModuleCRC21` dispatches EVM calls to registered CRC21 contract addresses without verifying that contract code still exists at the target address. Because the EVM returns `success=true` with empty return data when calling a codeless (e.g., selfdestructed) address, the `res.Failed()` guard in `CallModuleCRC21` does not fire. The surrounding bank operations in `ConvertCoinFromNativeToCRC21` execute successfully — permanently locking or burning the user's IBC vouchers or source-denom native tokens — while no CRC21 tokens are ever minted or transferred to the user.

---

### Finding Description

`CallModuleCRC21` packs the ABI call data, invokes `CallEVM` → `evmKeeper.ApplyMessage`, and checks only `res.Failed()`: [1](#0-0) 

`CallEVM` passes the call directly to `ApplyMessage` with no prior code-existence check: [2](#0-1) 

The EVM specification (and go-ethereum's `ApplyMessage`) returns `success=true` with empty return data when the destination address has no code. `res.Failed()` is therefore `false`, and `CallModuleCRC21` returns `([]byte{}, nil)` — silently succeeding.

`ConvertCoinFromNativeToCRC21` performs irreversible bank operations **before** or **after** the EVM call, with no existence guard:

**Non-source denom path** — coins are sent to the contract's Cosmos address first, then `mint_by_cronos_module` is called: [3](#0-2) 

**Source denom path** — coins are burned first, then `transfer_from_cronos_module` is called: [4](#0-3) 

The keeper does have an `ensureContractCode` helper: [5](#0-4) 

But it is called only at **registration time** inside `RegisterOrUpdateTokenMapping`: [6](#0-5) [7](#0-6) 

It is **never called** inside `ConvertCoinFromNativeToCRC21`, `ConvertCoinFromCRC21ToNative`, or `CallModuleCRC21`. After registration, if the external contract is selfdestructed, the denom→contract mapping persists in the KV store and all subsequent conversion calls proceed against a codeless address.

---

### Impact Explanation

**Non-source denom (IBC voucher) path:**
`bankKeeper.SendCoins` moves the user's IBC vouchers to the dead contract's Cosmos module address. The subsequent `mint_by_cronos_module` EVM call silently succeeds. No CRC21 tokens are minted. The IBC vouchers are permanently locked at an address controlled by no one.

**Source denom (`cronos0x…`) path:**
`bankKeeper.BurnCoins` permanently destroys the user's native source tokens. The subsequent `transfer_from_cronos_module` EVM call silently succeeds. No CRC21 tokens are released to the user. The tokens are unrecoverable.

Both outcomes constitute a **Critical** unauthorized balance/accounting change for IBC vouchers and CRC21/source-denom assets.

---

### Likelihood Explanation

The precondition is that an externally registered CRC21 contract (registered via `MsgUpdateTokenMapping` by the Cronos admin) is subsequently selfdestructed by its owner. The contract owner is an unprivileged EVM address distinct from the Cronos admin. Once the contract is dead, **any** user calling `MsgConvertVouchers` or triggering the IBC conversion middleware (`IBCConversionModule.OnRecvPacket` → `ConvertVouchersToEvmCoins`) against that denom will silently lose their assets. The victim is fully unprivileged and has no way to detect the danger before submitting the transaction. [8](#0-7) [9](#0-8) 

---

### Recommendation

Add a call to `ensureContractCode` (or `HasContractCode`) at the top of `ConvertCoinFromNativeToCRC21` and `ConvertCoinFromCRC21ToNative`, immediately after the contract address is resolved from the mapping and before any bank operation is performed. This mirrors the guard already present at registration time and ensures that a dead contract causes an explicit revert rather than a silent asset loss.

---

### Proof of Concept

1. Cronos admin calls `MsgUpdateTokenMapping` to register an external ERC20 contract `C` (which has a `selfdestruct` function) for IBC denom `D`. `ensureContractCode` passes at this point.
2. The owner of `C` calls `selfdestruct` on `C`. The EVM state for `C` is cleared; the Cronos KV store mapping `D → C` remains intact.
3. Alice holds 1000 `D` IBC vouchers and calls `MsgConvertVouchers{address: alice, coins: [{denom: D, amount: 1000}]}`.
4. `ConvertVouchersToEvmCoins` → `ConvertCoinFromNativeToCRC21` (non-source path):
   - `GetContractByDenom(D)` returns `C` (stale mapping, no existence check).
   - `bankKeeper.SendCoins(alice → C_cosmos, 1000 D)` succeeds — Alice's vouchers are now at `C`'s Cosmos address.
   - `CallModuleCRC21(C, "mint_by_cronos_module", alice, 1000)` → `ApplyMessage` on codeless address → `success=true`, empty return → `res.Failed() == false` → returns `nil`.
5. `MsgConvertVouchers` returns success. Alice has 0 IBC vouchers and 0 CRC21 tokens. The 1000 `D` vouchers are permanently locked at the dead contract's Cosmos address. [10](#0-9) [11](#0-10)

func (k Keeper) CallEVM(ctx sdk.Context, to *common.Address, data []byte, value *big.Int, gasLimit uint64) (*core.Message, *evmtypes.EVMResult, error) {
	nonce := k.evmKeeper.GetNonce(ctx, types.EVMModuleAddress)
	msg := &core.Message{
		From:            types.EVMModuleAddress,
		To:              to,
		Nonce:           nonce,
		Value:           value, // amount
		GasLimit:        gasLimit,
		GasPrice:        big.NewInt(0),
		GasFeeCap:       nil,
		GasTipCap:       nil, // gasPrice
		Data:            data,
		AccessList:      nil, // accessList
		SkipNonceChecks: false,
	}
	ret, err := k.evmKeeper.ApplyMessage(ctx, msg, nil, true)
	if err != nil {
		return nil, nil, err
	}

	// if the call is from an precompiled contract call, then re-emit the logs into the original stateDB.
	if stateDB, ok := ctx.Value(statedb.StateDBContextKey).(vm.StateDB); ok {
		for _, l := range ret.Logs {
			stateDB.AddLog(l.ToEthereum())
		}
	}

	return msg, ret, nil
}

// CallModuleCRC21 call a method of ModuleCRC21 contract
func (k Keeper) CallModuleCRC21(ctx sdk.Context, contract common.Address, method string, args ...interface{}) ([]byte, error) {
	data, err := types.ModuleCRC21Contract.ABI.Pack(method, args...)
	if err != nil {
		return nil, err
	}
	_, res, err := k.CallEVM(ctx, &contract, data, big.NewInt(0), DefaultGasCap)
	if err != nil {
		return nil, err
	}
	if res.Failed() {
		return nil, fmt.Errorf("call contract failed: %s, %s, %s", contract.Hex(), method, res.Ret)
	}
	return res.Ret, nil
}

func (k Keeper) ConvertCoinFromNativeToCRC21(ctx sdk.Context, sender common.Address, coin sdk.Coin, autoDeploy bool) error {
	if !types.IsValidCoinDenom(coin.Denom) {
		return fmt.Errorf("coin %s is not supported for conversion", coin.Denom)
	}
	var err error
	// external contract is returned in preference to auto-deployed ones
	contract, found := k.GetContractByDenom(ctx, coin.Denom)
	if !found {
		if !autoDeploy {
			return fmt.Errorf("no contract found for the denom %s", coin.Denom)
		}
		contract, err = k.DeployModuleCRC21(ctx, coin.Denom)
		if err != nil {
			return err
		}
		if err = k.SetAutoContractForDenom(ctx, coin.Denom, contract); err != nil {
			return err
		}

		k.Logger(ctx).Info(fmt.Sprintf("contract address %s created for coin denom %s", contract.String(), coin.Denom))
	}

	isSource := types.IsSourceCoin(coin.Denom)
	coins := sdk.NewCoins(coin)
	if isSource {
		// burn coins
		err = k.bankKeeper.SendCoinsFromAccountToModule(ctx, sdk.AccAddress(sender.Bytes()), types.ModuleName, sdk.NewCoins(coin))
		if err != nil {
			return err
		}
		err = k.bankKeeper.BurnCoins(ctx, types.ModuleName, coins)
		if err != nil {
			return err
		}
		// unlock crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "transfer_from_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	} else {
		// send coins to contract address
		err = k.bankKeeper.SendCoins(ctx, sdk.AccAddress(sender.Bytes()), sdk.AccAddress(contract.Bytes()), coins)
		if err != nil {
			return err
		}
		// mint crc tokens
		_, err = k.CallModuleCRC21(ctx, contract, "mint_by_cronos_module", sender, coin.Amount.BigInt())
		if err != nil {
			return err
		}
	}

	return nil
}

func (k Keeper) ensureContractCode(ctx sdk.Context, contract common.Address) error {
	if contract.Big().Cmp(big.NewInt(256)) < 0 {
		return errors.Wrapf(sdkerrors.ErrInvalidAddress,
			"crc21 contract must not be in precompile range: %s", contract.Hex())
	}
	resp, err := k.evmKeeper.Code(ctx, &evmtypes.QueryCodeRequest{
		Address: contract.Hex(),
	})
	if err != nil {
		return errors.Wrapf(sdkerrors.ErrInvalidAddress, "failed to query contract code (%s): %v", contract.Hex(), err)
	}
	if resp == nil || len(resp.Code) == 0 {
		return errors.Wrapf(sdkerrors.ErrInvalidRequest, "no contract code at address (%s)", contract.Hex())
	}
	return nil
}
