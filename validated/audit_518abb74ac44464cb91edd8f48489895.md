### Title
`get_compiled_class_hash` Missing `state_number` Guard Returns Future-Block Hash in Historical Simulation — (`crates/apollo_rpc_execution/src/state_reader.rs`)

---

### Summary

`ExecutionStateReader::get_compiled_class_hash` fetches the block at which a class was declared via `get_class_definition_block_number`, then unconditionally reads the compiled class hash from that block's state diff — without checking whether that block is at or before `self.state_number`. When a user calls a public RPC simulation/trace/fee-estimation endpoint at a historical block N, and a Sierra class was declared at a later block B > N, the function returns the real `CompiledClassHash` instead of `CompiledClassHash::default()`, corrupting the simulation's state diff output.

---

### Finding Description

The function at lines 163–208 of `crates/apollo_rpc_execution/src/state_reader.rs`:

```
get_class_definition_block_number(&class_hash)
  → Some(block_number)          // block_number may be > state_number
  → get_state_diff(block_number) // reads from the FUTURE block
  → returns compiled_class_hash  // wrong: class not yet declared at state_number
``` [1](#0-0) 

There is no guard equivalent to the one present in `get_contract_class` in `execution_utils.rs`:

```rust
Some(block_number) if state_number.is_before(block_number) => return Ok(None),
``` [2](#0-1) 

And the one in `get_class_definition_at` in `apollo_storage`:

```rust
if state_number.is_before(block_number) {
    return Ok(None);
}
``` [3](#0-2) 

The existing test in `state_reader_test.rs` line 186 does not catch this because `compiled_class_hash0` is set to `CompiledClassHash(StarkHash::default())`, which equals `CompiledClassHash::default()` — so the assertion passes whether the bug fires or not: [4](#0-3) [5](#0-4) 

---

### Impact Explanation

`ExecutionStateReader` is the `StateReader` implementation passed to `CachedState` in `execute_transactions`, which backs all RPC simulation, fee estimation, and trace calls: [6](#0-5) 

During execution of a `Declare` transaction in simulation, blockifier calls `CachedState::get_compiled_class_hash` to record the initial compiled class hash value in the state diff: [7](#0-6) 

When the queried `state_number` is before the block where the class was declared, the function should return `CompiledClassHash::default()` (the sentinel for "not yet declared / Cairo 0"). Instead it returns the real hash H from the future block B. This corrupts the `class_hash_to_compiled_class_hash` map in the `induced_state_diff` returned by the simulation, producing an authoritative-looking wrong value for any caller of `starknet_simulateTransactions`, `starknet_estimateFee`, or `starknet_traceTransaction` at a historical block.

---

### Likelihood Explanation

The trigger is straightforward: any unprivileged user can call a public RPC simulation endpoint with a `block_id` that is numerically less than the block in which a Sierra class was declared. No special privileges are required. The mainnet chain has thousands of declared classes, making the precondition trivially satisfiable.

---

### Recommendation

Add the missing temporal guard immediately after obtaining `block_number` from `get_class_definition_block_number`:

```rust
let Some(block_number) = maybe_block_number else {
    return Ok(CompiledClassHash::default());
};

// NEW: class was declared after the queried state; treat as undeclared.
if self.state_number.is_before(block_number) {
    return Ok(CompiledClassHash::default());
}
```

This mirrors the existing guard in `get_contract_class` and `get_class_definition_at`. The existing test in `state_reader_test.rs` should also be updated to use a non-default `compiled_class_hash0` so the regression is detectable.

---

### Proof of Concept

Scenario using the existing test infrastructure:

1. Write a Sierra class at block 10 with `compiled_class_hash = CompiledClassHash(Felt::from(42u64))` (non-default).
2. Construct an `ExecutionStateReader` with `state_number = StateNumber::unchecked_right_after_block(BlockNumber(5))`.
3. Call `state_reader.get_compiled_class_hash(class_hash)`.
4. **Expected**: `Ok(CompiledClassHash::default())` — class not yet declared at block 5.
5. **Actual (current code)**: `Ok(CompiledClassHash(Felt::from(42u64)))` — the hash from block 10 is returned, violating the state-number invariant.

The same incorrect hash then propagates into the `induced_state_diff` of any `starknet_simulateTransactions` call targeting a block before the class declaration, producing a wrong `class_hash_to_compiled_class_hash` entry in the simulation response. [8](#0-7)

    fn get_compiled_class_hash(&self, class_hash: ClassHash) -> StateResult<CompiledClassHash> {
        if let Some(pending_data) = &self.maybe_pending_data {
            for DeclaredClassHashEntry { class_hash: other_class_hash, compiled_class_hash } in
                &pending_data.declared_classes
            {
                if class_hash == *other_class_hash {
                    return Ok(*compiled_class_hash);
                }
            }
        }

        let maybe_block_number = self
            .storage_reader
            .begin_ro_txn()
            .map_err(storage_err_to_state_err)?
            .get_state_reader()
            .map_err(storage_err_to_state_err)?
            .get_class_definition_block_number(&class_hash)
            .map_err(storage_err_to_state_err)?;

        // Cairo 0 classes (and undeclared classes) do not have a compiled class hash.
        // According to the trait, return the default value.
        let Some(block_number) = maybe_block_number else {
            return Ok(CompiledClassHash::default());
        };

        let state_diff = self
            .storage_reader
            .begin_ro_txn()
            .map_err(storage_err_to_state_err)?
            .get_state_diff(block_number)
            .map_err(storage_err_to_state_err)?
            .ok_or(StateError::StateReadError(format!(
                "Inner storage error. Missing state diff at block {block_number}."
            )))?;

        let compiled_class_hash = state_diff
            .class_hash_to_compiled_class_hash
            .get(&class_hash)
            .ok_or(StateError::StateReadError(format!(
                "Inner storage error. Missing class declaration at block {block_number}, class \
                 {class_hash}."
            )))?;

        Ok(*compiled_class_hash)
    }
