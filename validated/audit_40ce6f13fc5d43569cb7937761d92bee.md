### Title
Gateway Admits Transactions with Arbitrarily Low L2 Gas Price During Genesis Phase Due to `GenesisFixedBlockStateReader` Returning `BlockInfo::default()` with `l2_gas_price = 1` — (`crates/apollo_gateway/src/sync_state_reader.rs`)

---

### Summary

When no committed blocks exist yet (genesis/bootstrap phase), `GenesisFixedBlockStateReader::get_block_info()` returns `BlockInfo::default()`, whose `strk_gas_prices.l2_gas_price` is `NonzeroGasPrice::MIN = 1 fri`. The gateway's stateful `validate_resource_bounds` check and the blockifier pre-validation inside `run_validate_entry_point` both consume this value as the "previous block L2 gas price." Because the actual first block will be built with an L2 gas price on the order of `VersionedConstants::latest_constants().min_gas_price` (~10⁹–10¹⁰ fri), any transaction whose `l2_gas.max_price_per_unit` is ≥ 1 fri passes gateway admission but will be rejected at execution time with `MaxGasPriceTooLow`. This is the direct Sequencer analog of the Elys `LastUpdatedBlock = 0` bug: a "last reference value" field is initialized to the absolute minimum instead of the correct current value, causing the downstream threshold check to be trivially bypassable.

---

### Finding Description

**Root cause — `GenesisFixedBlockStateReader`**

When `SyncStateReaderFactory::get_blockifier_state_reader_and_gateway_fixed_block_from_latest_block` is called and `get_latest_block_number()` returns `None`, the factory returns a `GenesisFixedBlockStateReader`:

```rust
// crates/apollo_gateway/src/sync_state_reader.rs  lines 531-537
let Some(latest_block_number) = latest_block_number else {
    info!("No blocks found yet; using genesis state readers for bootstrap transactions.");
    return Ok((GenesisStateReader.into(), GenesisFixedBlockStateReader.into()));
};
```

`GenesisFixedBlockStateReader::get_block_info()` unconditionally returns `BlockInfo::default()`:

```rust
// crates/apollo_gateway/src/sync_state_reader.rs  lines 374-378

    // Return a genesis block info with default/minimal values.
    // These values allow bootstrap transactions to pass validation.
    Ok(BlockInfo::default())
}
```

`NonzeroGasPrice::default()` is `NonzeroGasPrice::MIN = GasPrice(1)`:

```rust
// crates/starknet_api/src/block.rs  lines 556-559

    fn default() -> Self {
        Self::MIN   // = GasPrice(1)
    }
}
```

The test at line 450–457 of `sync_state_reader_test.rs` confirms all six gas-price fields in the genesis `BlockInfo` equal `1`.

**Path 1 — `validate_resource_bounds` threshold collapses to 1**

```rust

```

With `min_gas_price_percentage = 100` (the default):

```rust

    * previous_block_l2_gas_price.get().0)   // 100/100 * 1 = 1

```

**Path 2 — blockifier pre-validation inside `run_validate_entry_point` also uses price = 1**

```rust
// crates/apollo_gateway/src/stateful_transaction_validator.rs  lines 323-330
let mut block_info = self.gateway_fixed_block_state_reader.get_block_info().await?;
block_info.block_number = block_info.block_number.unchecked_next();
let block_context = BlockContext::new(
    block_info,          // gas prices all = 1
    self.chain_info.clone(),
    versioned_constants,
    BouncerConfig::max(),
);
```

`perform_pre_validation_stage → check_fee_bounds` compares `max_price_per_unit` against the block-context gas price (1), so a transaction with `max_price_per_unit = 1` passes the blockifier check too.

**Divergence at execution time**

When the first block is actually built, `SequencerConsensusContext::set_height_and_round` initialises `l2_gas_price` to `max(current, get_min_gas_price_for_height(...))`, which falls back to `VersionedConstants::latest_constants().min_gas_price` — a value in the Gwei range (~10⁹–10¹⁰ fri). The batcher's blockifier then runs `check_fee_bounds` against this real price, and every transaction whose `max_price_per_unit` is below it is rejected with `MaxGasPriceTooLow`.

---

### Impact Explanation

**Matching scope:** *High — Mempool/gateway/RPC admission accepts invalid transactions before sequencing.*

During the genesis window (before block 0 is committed), any unprivileged user can submit `AllResources` invoke/declare/deploy-account transactions with `l2_gas.max_price_per_unit = 1`. All such transactions pass both the gateway L2-gas-price threshold check and the blockifier pre-validation, are forwarded to the mempool, and are queued for the first block. At execution time they are rejected with `MaxGasPriceTooLow`, wasting batcher resources and potentially filling the mempool with permanently-unexecutable transactions, delaying or displacing legitimate transactions.

---

### Likelihood Explanation

The genesis window is short but deterministic and observable on-chain (no committed blocks). An attacker monitoring the node's RPC can detect the window and submit a burst of low-price transactions before block 0 is sealed. No special privilege is required; any RPC caller can trigger the path.

---

### Recommendation

`GenesisFixedBlockStateReader::get_block_info()` should return a `BlockInfo` whose gas prices are set to the actual protocol minimum rather than `NonzeroGasPrice::MIN`:

```rust
async fn get_block_info(&self) -> StarknetResult<BlockInfo> {
    let min_price = VersionedConstants::latest_constants().min_gas_price;
    let nonzero_min = NonzeroGasPrice::new(min_price)
        .unwrap_or(NonzeroGasPrice::MIN);
    let prices = GasPriceVector {
        l1_gas_price: nonzero_min,
        l1_data_gas_price: nonzero_min,
        l2_gas_price: nonzero_min,
    };
    Ok(BlockInfo {
        gas_prices: GasPrices {
            eth_gas_prices: prices,
            strk_gas_prices: prices,
        },
        ..BlockInfo::default()
    })
}
```

Alternatively, `validate_resource_bounds` can explicitly skip the check when the fixed-block reader is the genesis variant (matching the intent of the existing comment *"Skip this validation during the systems bootstrap phase"*).

---

### Proof of Concept

1. Start the sequencer with an empty storage (`get_latest_block_number()` → `None`).
2. Submit an `InvokeV3` transaction with `resource_bounds.l2_gas.max_price_per_unit = 1`.
3. **Gateway path:** `validate_resource_bounds` fetches `previous_block_l2_gas_price = 1` from `GenesisFixedBlockStateReader`; threshold = `1 * 100/100 = 1`; `1 < 1` is false → check passes.
4. **Blockifier path:** `run_validate_entry_point` builds a `BlockContext` with `l2_gas_price = 1`; `check_fee_bounds` compares `max_price_per_unit (1) < actual_gas_price (1)` → false → passes.
5. Transaction is forwarded to the mempool and accepted.
6. When the first block is built, `SequencerConsensusContext` initialises `l2_gas_price` to `VersionedConstants::latest_constants().min_gas_price` (~10⁹ fri). The batcher's blockifier runs `check_fee_bounds`: `1 < 10⁹` → `MaxGasPriceTooLow` → transaction is rejected at execution time.

```rust
    // TODO(Arni): Consider running this validation for all gas prices.
    fn validate_tx_l2_gas_price_within_threshold(
        &self,
        tx_resource_bounds: ValidResourceBounds,
        previous_block_l2_gas_price: NonzeroGasPrice,
    ) -> StatefulTransactionValidatorResult<()> {
        match tx_resource_bounds {
            ValidResourceBounds::AllResources(tx_resource_bounds) => {
                let tx_l2_gas_price = tx_resource_bounds.l2_gas.max_price_per_unit;
                let gas_price_threshold_multiplier =
                    Ratio::new(self.config.min_gas_price_percentage.into(), 100_u128);
                let threshold = (gas_price_threshold_multiplier
                    * previous_block_l2_gas_price.get().0)
                    .to_integer();
                if tx_l2_gas_price.0 < threshold {
                    return Err(StarknetError {
                        // We didn't have this kind of an error.
                        code: StarknetErrorCode::UnknownErrorCode(
                            "StarknetErrorCode.GAS_PRICE_TOO_LOW".to_string(),
                        ),
                        message: format!(
                            "Transaction L2 gas price {tx_l2_gas_price} is below the required \
                             threshold {threshold}.",
                        ),
                    });
                }
            }
            ValidResourceBounds::L1Gas(_) => {
                // No validation required for legacy transactions.
            }
        }
        Ok(())
    }
```

```rust
pub fn get_min_gas_price_for_height(
    height: BlockNumber,
    min_l2_gas_price_per_height: &[PricePerHeight],
) -> GasPrice {
    let fallback_min_gas_price = VersionedConstants::latest_constants().min_gas_price;
    min_l2_gas_price_per_height
        .iter()
        .rev()
        .find(|e| e.height <= height.0)
        .map(|e| GasPrice(e.price))
        .unwrap_or(fallback_min_gas_price)
}
```
