### Title
Pool Member Bypasses `exit_wait_window` via Expired Staker `unstake_time` in `compute_unpool_time` — (File: src/staking/objects.cairo)

---

### Summary

When a staker has called `unstake_intent` and the `exit_wait_window` has already elapsed, any pool member can call `exit_delegation_pool_intent` and immediately call `exit_delegation_pool_action` in the same block, bypassing their own mandatory `exit_wait_window`. The root cause is in `compute_unpool_time`, which returns `Time::now()` (instead of `Time::now() + exit_wait_window`) when the staker's `unstake_time` is already in the past.

---

### Finding Description

`compute_unpool_time` in `src/staking/objects.cairo` is called by `update_undelegate_intent_value` in `src/staking/staking.cairo` every time a pool member submits an exit intent while the staker is still registered (i.e., before `unstake_action` is called). Its logic is:

```cairo
fn compute_unpool_time(
    self: @InternalStakerInfoLatest, exit_wait_window: TimeDelta,
) -> Timestamp {
    if let Option::Some(unstake_time) = *self.unstake_time {
        return max(unstake_time, Time::now());   // ← vulnerable branch
    }
    Time::now().add(delta: exit_wait_window)
}
``` [1](#0-0) 

When the staker has already called `unstake_intent`, `staker_info.unstake_time` is `Some(T)` where `T = Time::now_at_intent + exit_wait_window`. Once wall-clock time advances past `T`, `max(T, Time::now()) = Time::now()`. The resulting `unpool_time` stored in `pool_exit_intents` is therefore the **current block timestamp**, not `Time::now() + exit_wait_window`.

The pool member's action gate in `exit_delegation_pool_action` only checks:

```cairo
assert!(Time::now() >= unpool_time, "{}", GenericError::INTENT_WINDOW_NOT_FINISHED);
``` [2](#0-1) 

Because `unpool_time` was set to `Time::now()` at intent time, this assertion passes immediately — even in the same block as the intent call.

The full call chain that reaches the vulnerable code:

1. `Pool::exit_delegation_pool_intent` → `undelegate_from_staking_contract_intent` [3](#0-2) 

2. `undelegate_from_staking_contract_intent` calls `staking_pool_dispatcher.remove_from_delegation_pool_intent` [4](#0-3) 

3. `Staking::remove_from_delegation_pool_intent` calls `update_undelegate_intent_value` [5](#0-4) 

4. `update_undelegate_intent_value` calls `staker_info.compute_unpool_time` [6](#0-5) 

---

### Impact Explanation

The `exit_wait_window` (default 1 week, max 12 weeks) is the protocol's primary safety delay between a withdrawal intent and the actual release of funds. [7](#0-6) 

A pool member who times their `exit_delegation_pool_intent` call to land after the staker's `unstake_time` has expired can withdraw their delegated stake with **zero waiting time**, completely circumventing the protocol's mandatory lock-up period. This violates the protocol's stated invariant and undermines any future slashing or protocol-pause mechanism that relies on the window being enforced.

**Allowed impact matched**: Medium — Griefing with no profit motive but damage to users or protocol (protocol safety invariant broken; exit_wait_window rendered ineffective for pool members in this state).

---

### Likelihood Explanation

The precondition is that the staker has called `unstake_intent` and the `exit_wait_window` has elapsed without `unstake_action` being called yet. This is a realistic window: `unstake_action` is permissionless but requires an external caller; there is no on-chain enforcement that it is called immediately after `unstake_time` passes. Any pool member who monitors the chain can observe the staker's `unstake_time` from the `StakerExitIntent` event and time their intent call accordingly. No privileged access is required.

---

### Recommendation

Replace the `max(unstake_time, Time::now())` shortcut with a value that always enforces the full `exit_wait_window` from the pool member's perspective:

```cairo
fn compute_unpool_time(
    self: @InternalStakerInfoLatest, exit_wait_window: TimeDelta,
) -> Timestamp {
    let base = if let Option::Some(unstake_time) = *self.unstake_time {
        max(unstake_time, Time::now())
    } else {
        Time::now()
    };
    // Always add the full exit_wait_window from the pool member's call time.
    max(base, Time::now().add(delta: exit_wait_window))
}
```

This ensures the pool member always waits at least `exit_wait_window` from the moment they submit their own intent, regardless of the staker's unstake timeline.

---

### Proof of Concept

1. Staker calls `unstake_intent()` at time `T0`. `staker_info.unstake_time = T0 + 1 week`. [8](#0-7) 

2. Time advances to `T1 > T0 + 1 week`. Staker has not yet called `unstake_action`.

3. Pool member calls `exit_delegation_pool_intent(amount)`.
   - `undelegate_from_staking_contract_intent` is called; staker is still active (`staker_removed == false`), so it calls `remove_from_delegation_pool_intent`.
   - `compute_unpool_time` executes: `max(T0 + 1 week, T1) = T1 = Time::now()`.
   - `pool_exit_intents[key].unpool_time = T1`. [1](#0-0) 

4. In the **same block** (same `Time::now() = T1`), pool member calls `exit_delegation_pool_action(pool_member)`.
   - Check: `T1 >= T1` → passes.
   - Funds transferred to pool member immediately, with zero waiting time. [9](#0-8)

        fn undelegate_from_staking_contract_intent(
            self: @ContractState, pool_member: ContractAddress, amount: Amount,
        ) -> Timestamp {

                // Don't allow intent if an intent is already in progress and the staker is erased.
                // Avoid the following flow:
                // 1. Member intent - moves member balance from the staker's `pool_amount` to
                // `UndelegateIntentKey`.
                // 2. Staker intent.
                // 3. Staker action - transfer the `pool_amount` from the staker to the pool

                    "{}",

                );

            let staker_address = self.staker_address.read();
            staking_pool_dispatcher
                .remove_from_delegation_pool_intent(
                    :staker_address, identifier: pool_member.into(), :amount,
                )
        }

        fn unstake_intent(ref self: ContractState) -> Timestamp {
            // Prerequisites and asserts.
            self.general_prerequisites();
            let staker_address = get_caller_address();
            let mut staker_info = self.internal_staker_info(:staker_address);
            assert!(staker_info.unstake_time.is_none(), "{}", Error::UNSTAKE_IN_PROGRESS);

            // Set the unstake time.
            let unstake_time = Time::now().add(delta: self.exit_wait_window.read());
            staker_info.unstake_time = Option::Some(unstake_time);
            self.write_staker_info(:staker_address, :staker_info);

            // Write the unstake intent epoch.
            self.staker_unstake_intent_epoch.write(staker_address, self.get_epoch_plus_k());

            // Write off the delegated stake from the total stake.
            for (pool_contract, token_address) in self
                .staker_pool_info
                .entry(staker_address)
                .pools {
                let amount = self.get_delegated_balance(:staker_address, :pool_contract);
                self.remove_from_total_stake(:token_address, :amount);
                let decimals = self.get_token_decimals(:token_address);
                self
                    .emit(
                        Events::StakeDelegatedBalanceChanged {
                            staker_address,
                            token_address,
                            old_delegated_stake: amount.to_native_amount(:decimals),
                            new_delegated_stake: Zero::zero(),
                        },
                    );
            }
            // Write off the self stake from the total stake.
            let old_self_stake = self.get_own_balance(:staker_address);
            self.remove_from_total_stake(token_address: STRK_TOKEN_ADDRESS, amount: old_self_stake);

            // Emit events.
            self.emit(Events::StakerExitIntent { staker_address, exit_timestamp: unstake_time });
            self
                .emit(
                    Events::StakeOwnBalanceChanged {
                        staker_address,
                        old_self_stake: old_self_stake.to_strk_native_amount(),
                        new_self_stake: Zero::zero(),
                    },
                );
            unstake_time
        }
