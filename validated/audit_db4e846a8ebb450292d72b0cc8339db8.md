### Title
Blocklist Enforcement Bypassed for Native Cosmos SDK Message Recipients — (`app/proposal.go`, `app/block_address.go`)

### Summary
Cronos enforces a validator-managed blocklist at both the mempool ante-handler level (`BlockAddressesDecorator`) and the consensus proposal level (`ValidateTransaction`). Both enforcement points check the transaction **signer** and, for `MsgEthereumTx`, the EVM **destination address**. However, neither enforcement point inspects the recipient address embedded in native Cosmos SDK messages (e.g., `MsgSend.to_address`). An unprivileged, non-blocked address can therefore transfer CRO or IBC vouchers directly to a blocked address by submitting a native Cosmos SDK bank transfer, fully bypassing the blocklist.

### Finding Description

`ValidateTransaction` in `app/proposal.go` iterates over `tx.GetMsgs()` and only performs a destination-address check for `*evmtypes.MsgEthereumTx`:

```go
for _, msg := range tx.GetMsgs() {
    msgEthTx, ok := msg.(*evmtypes.MsgEthereumTx)
    if ok {
        ethTx := msgEthTx.AsTransaction()
        // check the destination address
        if ethTx.To() != nil {
            ...
            if _, ok := h.blocklist[encoded]; ok {
                return fmt.Errorf("destination address is blocked: %s", encoded)
            }
        }
        ...
    }
}
``` [1](#0-0) 

For any non-EVM message type, only the **signer** is checked:

```go
for _, signer := range signers {
    ...
    if _, ok := h.blocklist[encoded]; ok {
        return fmt.Errorf("signer is blocked: %s", encoded)
    }
}
``` [2](#0-1) 

The identical gap exists in the mempool ante-handler `BlockAddressesDecorator.AnteHandle`, which also only checks signers and `MsgEthereumTx.To()`, with no inspection of native message recipient fields. [3](#0-2) 

The Cosmos SDK `MsgSend` message carries a `to_address` field that is never examined by either enforcement point. A non-blocked sender can craft:

```
MsgSend { from_address: <non-blocked>, to_address: <blocked>, amount: [...] }
```

This transaction passes all blocklist checks (signer is not blocked, no `MsgEthereumTx` present) and is included in a block, crediting the blocked address with CRO or IBC vouchers.

### Impact Explanation

The blocklist is a security control (evidenced by the explicit addition of EVM destination checks in [#1922](https://github.com/crypto-org-chain/cronos/pull/1922) and [#1924](https://github.com/crypto-org-chain/cronos/pull/1924)) intended to prevent blocked addresses from participating in asset flows. Bypassing it via native `MsgSend` allows a blocked address to receive CRO or IBC voucher balances, directly undermining the blocklist's purpose. This maps to the allowed High impact: **Bypass of Cronos admin/block-list authorization checks**.

### Likelihood Explanation

The attack path requires no special privileges: any non-blocked address can submit a standard Cosmos SDK bank `MsgSend` transaction. The Cosmos SDK bank module is always available on Cronos. The bypass is unconditional — it does not depend on chain state, timing, or any other condition.

### Recommendation

In `ValidateTransaction` (and symmetrically in `BlockAddressesDecorator.AnteHandle`), after the signer loop, add a type-switch over all message types that carry a recipient address. At minimum, handle `*banktypes.MsgSend` and `*banktypes.MsgMultiSend`:

```go
case *banktypes.MsgSend:
    encoded, err := h.addressCodec.BytesToString([]byte(m.ToAddress))
    if err == nil {
        if _, ok := h.blocklist[encoded]; ok {
            return fmt.Errorf("recipient address is blocked: %s", encoded)
        }
    }
case *banktypes.MsgMultiSend:
    for _, output := range m.Outputs {
        encoded, err := h.addressCodec.BytesToString([]byte(output.Address))
        if err == nil {
            if _, ok := h.blocklist[encoded]; ok {
                return fmt.Errorf("recipient address is blocked: %s", encoded)
            }
        }
    }
```

Also consider `MsgTransferTokens.To` for completeness. [4](#0-3) 

### Proof of Concept

1. Admin stores a blocklist containing `blocked_addr` via `MsgStoreBlockList`.
2. Validators decrypt and load the blocklist; `ValidateTransaction` now rejects any tx signed by `blocked_addr`.
3. Attacker (non-blocked `attacker_addr`) submits:
   ```
   MsgSend { from_address: attacker_addr, to_address: blocked_addr, amount: [1000000basetcro] }
   ```
4. `ValidateTransaction` checks signers → `attacker_addr` not in blocklist → passes. Iterates msgs → not `MsgEthereumTx` → no destination check → passes.
5. Transaction is included in a block. `blocked_addr` balance increases by 1000000basetcro.
6. The blocklist has been bypassed: `blocked_addr` received CRO despite being on the blocklist. [1](#0-0) [5](#0-4)

func (h *ProposalHandler) ValidateTransaction(tx sdk.Tx, txBz []byte) error {
	if len(h.blocklist) == 0 {
		// fast path, accept all txs
		return nil
	}

	var err error
	if tx == nil {
		tx, err = h.TxDecoder(txBz)
		if err != nil {
			return err
		}
	}

	sigTx, ok := tx.(signing.SigVerifiableTx)
	if !ok {
		return fmt.Errorf("tx of type %T does not implement SigVerifiableTx", tx)
	}

	signers, err := sigTx.GetSigners()
	if err != nil {
		return err
	}
	for _, signer := range signers {
		encoded, err := h.addressCodec.BytesToString(signer)
		if err != nil {
			return fmt.Errorf("invalid bech32 address: %s, err: %w", signer, err)
		}
		if _, ok := h.blocklist[encoded]; ok {
			return fmt.Errorf("signer is blocked: %s", encoded)
		}
	}

	for _, msg := range tx.GetMsgs() {
		msgEthTx, ok := msg.(*evmtypes.MsgEthereumTx)
		if ok {
			ethTx := msgEthTx.AsTransaction()
			// check the destination address
			if ethTx.To() != nil {
				encoded, err := h.addressCodec.BytesToString(ethTx.To().Bytes())
				if err != nil {
					return fmt.Errorf("invalid bech32 address: %s, err: %w", ethTx.To(), err)
				}
				if _, ok := h.blocklist[encoded]; ok {
					return fmt.Errorf("destination address is blocked: %s", encoded)
				}
			}
			// check EIP-7702 authorisation list
			if ethTx.SetCodeAuthorizations() != nil {
				for _, auth := range ethTx.SetCodeAuthorizations() {
					addr, err := auth.Authority()
					if err == nil {
						encoded, err := h.addressCodec.BytesToString(addr.Bytes())
						if err != nil {
							return fmt.Errorf("invalid bech32 address: %s, err: %w", addr, err)
						}
						if _, ok := h.blocklist[encoded]; ok {
							return fmt.Errorf("signer is blocked: %s", encoded)
						}
					}
					// check the target address
					encoded, err := h.addressCodec.BytesToString(auth.Address.Bytes())
					if err != nil {
						return fmt.Errorf("invalid bech32 address: %s, err: %w", auth.Address, err)
					}
					if _, ok := h.blocklist[encoded]; ok {
						return fmt.Errorf("authorisation address is blocked: %s", encoded)
					}
				}
			}
		}
	}

	return nil
}

func (bad BlockAddressesDecorator) AnteHandle(ctx sdk.Context, tx sdk.Tx, simulate bool, next sdk.AnteHandler) (newCtx sdk.Context, err error) {
	if ctx.IsCheckTx() {
		if sigTx, ok := tx.(signing.SigVerifiableTx); ok {
			signers, err := sigTx.GetSigners()
			if err != nil {
				return ctx, err
			}
			for _, signer := range signers {
				if _, ok := bad.blockedMap[sdk.AccAddress(signer).String()]; ok {
					return ctx, fmt.Errorf("signer is blocked: %s", sdk.AccAddress(signer).String())
				}
			}
		}

		for _, msg := range tx.GetMsgs() {
			msgEthTx, ok := msg.(*evmtypes.MsgEthereumTx)
			if ok {
				ethTx := msgEthTx.AsTransaction()
				// check the destination address
				if ethTx.To() != nil {
					if _, ok := bad.blockedMap[sdk.AccAddress(ethTx.To().Bytes()).String()]; ok {
						return ctx, fmt.Errorf("destination address is blocked: %s", sdk.AccAddress(ethTx.To().Bytes()).String())
					}
				}
				// check EIP-7702 authorisation list
				if ethTx.SetCodeAuthorizations() != nil {
					for _, auth := range ethTx.SetCodeAuthorizations() {
						addr, err := auth.Authority()
						if err == nil {
							if _, ok := bad.blockedMap[sdk.AccAddress(addr.Bytes()).String()]; ok {
								return ctx, fmt.Errorf("signer is blocked: %s", addr.String())
							}
						}
						// check the target address
						if _, ok := bad.blockedMap[sdk.AccAddress(auth.Address.Bytes()).String()]; ok {
							return ctx, fmt.Errorf("authorisation address is blocked: %s", sdk.AccAddress(auth.Address.Bytes()).String())
						}
					}
				}
			}
		}

		admin := bad.getParams(ctx).CronosAdmin
		for _, msg := range tx.GetMsgs() {
			if blocklistMsg, ok := msg.(*types.MsgStoreBlockList); ok {
				if admin != blocklistMsg.From {
					return ctx, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
				}
			}
		}
	}
	return next(ctx, tx, simulate)
}
