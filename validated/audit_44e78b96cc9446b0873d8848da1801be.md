### Title
Stale KV-Store Permissions Survive `CronosAdmin` Clearance, Allowing Unauthorized Token-Mapping Corruption - (File: `x/cronos/keeper/permissions.go`, `x/cronos/keeper/msg_server.go`)

### Summary
`HasPermission` evaluates two independent conditions: the live `CronosAdmin` param **and** per-address permission bits stored in the KV store. When governance clears `CronosAdmin` to `""` to lock down all privileged operations, previously-permissioned addresses retain their `CanChangeTokenMapping` bits in the KV store. Because `UpdatePermissions` requires a non-empty admin to revoke those bits, the stored permissions become permanently irrevocable. Any address that was previously granted `CanChangeTokenMapping` can still call `MsgUpdateTokenMapping` and corrupt the denom→contract mapping used by every IBC/gravity token conversion.

### Finding Description

`HasPermission` in `x/cronos/keeper/permissions.go` checks two independent authorization paths:

```go
func (k Keeper) HasPermission(ctx sdk.Context, accounts []sdk.AccAddress, permissionsToCheck uint64) bool {
    admin := k.GetParams(ctx).CronosAdmin
    for _, account := range accounts {
        if admin == account.String() {   // path 1: live param
            return true
        }
        permission := k.GetPermissions(ctx, account)
        if permission&permissionsToCheck == permissionsToCheck { // path 2: KV store
            return true
        }
    }
    return false
}

The only way to revoke stored permissions is `UpdatePermissions`, which requires `admin == msg.From`:

```go
admin := k.Keeper.GetParams(ctx).CronosAdmin
// if admin is empty, no sender could be equal to it
if admin != msg.From {
    return nil, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")

When governance sets `CronosAdmin = ""` (the documented "no admin" state), `UpdatePermissions` becomes permanently uncallable — no address can equal `""`. Any address that previously received `CanChangeTokenMapping` via `SetPermissions` retains that bit in the KV store indefinitely.

}

### Impact Explanation

An attacker with a stale `CanChangeTokenMapping` bit can:
1. Call `MsgUpdateTokenMapping` with `contract = ""` to delete the legitimate denom→contract mapping for a live IBC/gravity token.
2. Call `MsgUpdateTokenMapping` again with a malicious contract address to install a new mapping.

From that point, every incoming IBC packet or gravity bridge transfer for that denom triggers `OnRecvVouchers` → `ConvertVouchersToEvmCoins` → `GetContractByDenom`, which now resolves to the malicious contract. The module calls `mint_by_cronos_module` on the malicious contract, permanently misdirecting all converted tokens. This is a direct, irreversible corruption of the denom/contract binding and IBC accounting state.

**Allowed impact matched:** "Corruption of token mappings, denom/contract binding, IBC channel/accounting state… with direct security impact" (High) and "Bypass of Cronos admin, governance authority, permission… authorization checks" (High).

### Likelihood Explanation

The precondition — governance clearing `CronosAdmin` while at least one address holds a stored permission — is operationally realistic. The ADR-009 design explicitly anticipates delegating `CanChangeTokenMapping` to external systems (e.g., automated relayers). A governance proposal to remove the admin (e.g., to freeze the system during an incident) would not automatically revoke those delegated permissions. The attacker needs no leaked keys, no validator compromise, and no cryptographic break; they only need to have been a previously-trusted operator.

### Recommendation

When `CronosAdmin` is set to `""` (or changed to a new address), the `UpdateParams` handler in `msg_server.go` should iterate over all stored permission entries and clear them, or alternatively `HasPermission` should short-circuit to `false` for the KV-store path whenever `CronosAdmin` is empty:

```go
// In HasPermission: only check stored permissions if an admin exists
if admin == "" {
    return false
}
```

A safer long-term fix is to require that `UpdateParams` atomically clears all stored permissions whenever `CronosAdmin` changes, mirroring the recommendation in the external report to clear pending state before locking down ownership. [7](#0-6) 

### Proof of Concept

1. Admin calls `MsgUpdatePermissions{From: admin, Address: attacker, Permissions: 1}` → `attacker` now has `CanChangeTokenMapping` in the KV store.
2. Governance passes `MsgUpdateParams` setting `CronosAdmin = ""`.
3. `UpdatePermissions` is now permanently uncallable (no address equals `""`).
4. Attacker calls `MsgUpdateTokenMapping{Sender: attacker, Denom: "gravity0x<token>", Contract: ""}` → deletes the legitimate mapping. `HasPermission` returns `true` via the KV-store path.
5. Attacker calls `MsgUpdateTokenMapping{Sender: attacker, Denom: "gravity0x<token>", Contract: "0x<malicious>"}` → installs malicious mapping.
6. All subsequent IBC receives for `gravity0x<token>` call `mint_by_cronos_module` on `0x<malicious>`, permanently misdirecting converted tokens. [8](#0-7) [9](#0-8)

func (k Keeper) HasPermission(ctx sdk.Context, accounts []sdk.AccAddress, permissionsToCheck uint64) bool {
	// case when no permission is needed
	if permissionsToCheck == 0 {
		return true
	}
	admin := k.GetParams(ctx).CronosAdmin
	for _, account := range accounts {
		if admin == account.String() {
			return true
		}
		permission := k.GetPermissions(ctx, account)
		if permission&permissionsToCheck == permissionsToCheck {
			return true
		}
	}

	return false

func (k msgServer) UpdateTokenMapping(goCtx context.Context, msg *types.MsgUpdateTokenMapping) (*types.MsgUpdateTokenMappingResponse, error) {
	ctx := sdk.UnwrapSDKContext(goCtx)

	// check permission
	if !k.HasPermission(ctx, msg.GetSigners(), CanChangeTokenMapping) {
		return nil, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
	}

	// msg is already validated
	if err := k.RegisterOrUpdateTokenMapping(ctx, msg); err != nil {
		return nil, err
	}
	return &types.MsgUpdateTokenMappingResponse{}, nil
}

func (k msgServer) UpdatePermissions(goCtx context.Context, msg *types.MsgUpdatePermissions) (*types.MsgUpdatePermissionsResponse, error) {
	ctx := sdk.UnwrapSDKContext(goCtx)
	admin := k.Keeper.GetParams(ctx).CronosAdmin
	// if admin is empty, no sender could be equal to it
	if admin != msg.From {
		return nil, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
	}
	acc, err := sdk.AccAddressFromBech32(msg.Address)
	if err != nil {
		return nil, err
	}
	k.SetPermissions(ctx, acc, msg.Permissions)

	return &types.MsgUpdatePermissionsResponse{}, nil

func (k Keeper) RegisterOrUpdateTokenMapping(ctx sdk.Context, msg *types.MsgUpdateTokenMapping) error {
	if types.IsSourceCoin(msg.Denom) {
		_, err := types.GetContractAddressFromDenom(msg.Denom)
		if err != nil {
			return err
		}

		if !common.IsHexAddress(msg.Contract) {
			return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
		}
		contract := common.HexToAddress(msg.Contract)
		if err := k.ensureContractCode(ctx, contract); err != nil {
			return err
		}
		if err := validateContractAddressForSourceDenom(msg.Denom, contract, true); err != nil {
			return err
		}

		if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
			return err
		}

		// check that the coin is registered, otherwise register it
		metadata, exist := k.bankKeeper.GetDenomMetaData(ctx, msg.Denom)
		if !exist {
			// create new metadata
			metadata = banktypes.Metadata{
				Base: msg.Denom,
				Name: msg.Denom,
			}
		}
		// update existing metadata
		metadata.Symbol = msg.Symbol
		metadata.Display = strings.ToLower(msg.Symbol)
		if msg.Decimal != 0 {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
				{
					Denom:    metadata.Display,
					Exponent: msg.Decimal,
				},
			}
		} else {
			metadata.DenomUnits = []*banktypes.DenomUnit{
				{
					Denom:    metadata.Base,
					Exponent: 0,
				},
			}
		}
		k.bankKeeper.SetDenomMetaData(ctx, metadata)
	} else {
		if len(msg.Contract) == 0 {
			// delete existing mapping
			k.DeleteExternalContractForDenom(ctx, msg.Denom)
		} else {
			if !common.IsHexAddress(msg.Contract) {
				return errors.Wrapf(sdkerrors.ErrInvalidAddress, "invalid contract address (%s)", msg.Contract)
			}
			// update the mapping
			contract := common.HexToAddress(msg.Contract)
			if err := k.ensureContractCode(ctx, contract); err != nil {
				return err
			}
			if err := k.SetExternalContractForDenom(ctx, msg.Denom, contract); err != nil {
				return err
			}
		}
	}

	return nil
}
