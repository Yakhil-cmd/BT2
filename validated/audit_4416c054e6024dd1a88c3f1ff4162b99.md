### Title
Any unprivileged caller can permanently freeze staker block rewards by front-running `update_rewards` with `disable_rewards: true` — (File: src/staking/staking.cairo)

### Summary
The `update_rewards` function in the Staking contract is publicly callable with no access control. Any caller can invoke it with `disable_rewards: true` for any staker, which advances the global `last_reward_block` to the current block without distributing rewards. Because only one call per block is permitted (enforced by the `REWARDS_ALREADY_UPDATED` guard), this permanently consumes the block's reward slot and freezes the staker's unclaimed yield for that block.

### Finding Description
`StakingRewardsManagerImpl::update_rewards` accepts a caller-supplied `disable_rewards: bool` parameter and enforces no caller restriction beyond `general_prerequisites`, which only checks that the contract is unpaused and the caller is non-zero. [1](#0-0) 

When called with `disable_rewards: true` the function:

1. Asserts `current_block_number > last_reward_block` (passes on the first call in a block).
2. Writes `last_reward_block = current_block_number`.
3. Returns **early**, skipping the entire `_update_rewards` path. [2](#0-1) 

`last_reward_block` is a **single global storage variable**, not per-staker: [3](#0-2) 

Because only one call per block is allowed across all stakers, an attacker who calls `update_rewards(any_staker, true)` in block B:

- Advances `last_reward_block` to B without distributing anything.
- Causes every subsequent `update_rewards(staker, false)` call in block B to revert with `REWARDS_ALREADY_UPDATED`.
- Permanently prevents `_update_rewards` from running for block B, so `staker_info.unclaimed_rewards_own` is never incremented and pool rewards are never forwarded. [4](#0-3) 

The `general_prerequisites` guard that is the only barrier: [5](#0-4) 

### Impact Explanation
This is a **permanent freezing of unclaimed yield**. Block rewards for the targeted block are never credited to `unclaimed_rewards_own` and never forwarded to delegation pools via `update_pool_rewards`. There is no recovery path — the block has passed and `last_reward_block` already records it as processed. An attacker who repeats this every block reduces a staker's (and all associated pool members') earned rewards to zero indefinitely.

### Likelihood Explanation
The entry path is fully unprivileged: any non-zero address can call `update_rewards(staker_address, true)` with no stake, no role, and no special permission. The only cost is gas. The attacker does not need to front-run in the strict mempool sense — submitting the transaction at the start of any block before the legitimate caller is sufficient. The attack is cheap, repeatable, and requires no coordination.

### Recommendation
Restrict who may call `update_rewards` with `disable_rewards: true`. Options include:

- Add an access-control check so only the attestation contract or a designated privileged operator can supply `disable_rewards: true`.
- Split the function: keep a public path that always distributes rewards, and expose a separate privileged function for the "mark block processed without rewards" case.
- At minimum, require `get_caller_address() == staker_address || get_caller_address() == staker_info.operational_address` so only the staker or their operator can invoke the function.

### Proof of Concept
1. Staker S is active and eligible for block rewards in block B.
2. Attacker A calls `update_rewards(S, disable_rewards: true)` in block B (or any block where `current_block_number > last_reward_block`).
3. `last_reward_block` is set to B; the function returns without executing `_update_rewards`.
4. S (or anyone else) calls `update_rewards(S, false)` in block B → reverts with `REWARDS_ALREADY_UPDATED`.
5. S's `unclaimed_rewards_own` is not incremented for block B; pool rewards for block B are not distributed to pool members.
6. Repeated every block, S and all pool members earn zero rewards indefinitely.

    impl StakingRewardsManagerImpl of IStakingRewardsManager<ContractState> {
        fn update_rewards(
            ref self: ContractState, staker_address: ContractAddress, disable_rewards: bool,
        ) {
            self.general_prerequisites();
            let current_block_number = starknet::get_block_number();
            assert!(
                current_block_number > self.last_reward_block.read(),
                "{}",
                Error::REWARDS_ALREADY_UPDATED,
            );

            // Assert staker exists and active.
            // Staker is considered to exist from the moment of `stake` (when `InternalStakerInfo`
            // struct is created) until the calling to `unstake_action` (when `InternalStakerInfo`
            // struct is deleted).
            // Staker remains active until the intent period begins, i.e. K epochs after
            // `unstake_intent` is called.
            let staker_info = self.internal_staker_info(:staker_address);
            let curr_epoch = self.get_current_epoch();
            assert!(
                self.is_staker_active(:staker_address, epoch_id: curr_epoch),
                "{}",
                Error::INVALID_STAKER,
            );

            let staker_pool_info = self.staker_pool_info.entry(staker_address).as_non_mut();
            let (staker_total_strk_balance, staker_total_btc_balance) = self
                .get_staker_total_strk_btc_balance_at_epoch(
                    :staker_address, :staker_pool_info, epoch_id: curr_epoch,
                );
            // Assert staker has non-zero balance.
            // Staker exists with zero balance for the first K epochs after `stake`, then the stake
            // becomes effective.
            assert!(staker_total_strk_balance.is_non_zero(), "{}", Error::INVALID_STAKER);

            // Update last block rewards.
            self.last_reward_block.write(current_block_number);

            if disable_rewards || self.is_pre_consensus() {
                return;
            }

            // Get current block data and update rewards.
            let reward_supplier_dispatcher = self.reward_supplier_dispatcher.read();
            let (strk_block_rewards, btc_block_rewards) = self
                .calculate_block_rewards(:reward_supplier_dispatcher, :curr_epoch);
            self
                ._update_rewards(
                    :staker_address,
                    strk_total_rewards: strk_block_rewards,
                    btc_total_rewards: btc_block_rewards,
                    strk_total_stake: staker_total_strk_balance,
                    btc_total_stake: staker_total_btc_balance,
                    :staker_info,
                    :staker_pool_info,
                    :reward_supplier_dispatcher,
                    :curr_epoch,
                );
        }
    }

        fn _update_rewards(
            ref self: ContractState,
            staker_address: ContractAddress,
            strk_total_rewards: Amount,
            btc_total_rewards: Amount,
            strk_total_stake: NormalizedAmount,
            btc_total_stake: NormalizedAmount,
            mut staker_info: InternalStakerInfoLatest,
            staker_pool_info: StoragePath<InternalStakerPoolInfoV2>,
            reward_supplier_dispatcher: IRewardSupplierDispatcher,
            curr_epoch: Epoch,
        ) {
            // Calculate self rewards.
            let staker_own_rewards = self
                .calculate_staker_own_rewards(
                    :staker_address, :strk_total_rewards, :strk_total_stake, :curr_epoch,
                );

            // Calculate pools rewards.
            let (commission_rewards, total_pools_rewards, pools_rewards_data) = if staker_pool_info
                .has_pool() {
                self
                    .calculate_staker_pools_rewards(
                        :staker_address,
                        :staker_pool_info,
                        :strk_total_rewards,
                        :strk_total_stake,
                        :btc_total_rewards,
                        :btc_total_stake,
                        :curr_epoch,
                    )
            } else {
                (Zero::zero(), Zero::zero(), array![])
            };

            // Update reward supplier.
            let staker_rewards = staker_own_rewards + commission_rewards;
            // Update total rewards.
            reward_supplier_dispatcher
                .update_unclaimed_rewards_from_staking_contract(
                    rewards: staker_rewards + total_pools_rewards,
                );
            // Claim pools rewards.
            claim_from_reward_supplier(
                :reward_supplier_dispatcher,
                amount: total_pools_rewards,
                token_dispatcher: strk_token_dispatcher(),
            );
            // Update staker rewards.
            staker_info.unclaimed_rewards_own += staker_rewards;

            // Update pools rewards.
            let pool_rewards_list = self.update_pool_rewards(:staker_address, :pools_rewards_data);
            // Emit event.
            self
                .emit(
                    Events::StakerRewardsUpdated {
                        staker_address, staker_rewards, pool_rewards: pool_rewards_list.span(),
                    },
                );

            // Write staker rewards to storage.
            self.write_staker_info(:staker_address, :staker_info);
        }
