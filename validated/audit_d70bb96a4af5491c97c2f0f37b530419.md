### Title
Unprivileged Caller Can Front-Run `update_rewards` With `disable_rewards: true`, Permanently Blocking Consensus Reward Distribution — (File: `src/staking/staking.cairo`)

---

### Summary

The `update_rewards` function in the Staking contract is callable by any unprivileged address and accepts a `disable_rewards` boolean parameter. When called with `disable_rewards: true`, it writes the current block number to the **global** `last_reward_block` storage slot without distributing any rewards. Because the guard `current_block_number > last_reward_block` is checked before the `disable_rewards` branch, any subsequent legitimate call in the same block reverts with `REWARDS_ALREADY_UPDATED`. An attacker can exploit this every block to permanently freeze all stakers' consensus-era unclaimed yield at zero cost beyond gas.

---

### Finding Description

`update_rewards` is part of `IStakingRewardsManager` and has no caller restriction beyond `general_prerequisites` (not paused, caller non-zero). [1](#0-0) 

The function first checks the global `last_reward_block`: [2](#0-1) 

It then unconditionally writes the current block number to `last_reward_block` **before** branching on `disable_rewards`: [3](#0-2) 

`last_reward_block` is a single global slot, not per-staker: [4](#0-3) 

When `disable_rewards` is `true`, the function returns immediately after updating `last_reward_block`, skipping the entire reward calculation and distribution path: [5](#0-4) 

Because `last_reward_block` is now equal to the current block, every other call to `update_rewards` in the same block — for any staker — reverts with `REWARDS_ALREADY_UPDATED`. The reward slot for that block is permanently consumed with no rewards distributed.

---

### Impact Explanation

An attacker who calls `update_rewards(any_active_staker, disable_rewards: true)` once per block:

1. Consumes the global per-block reward slot.
2. Causes every legitimate `update_rewards` call in that block to revert.
3. Ensures no staker ever accumulates consensus-era block rewards.

This constitutes **permanent freezing of unclaimed yield** for all stakers in the consensus rewards phase. The staking contract's `_update_rewards` path — which credits `staker_info.unclaimed_rewards_own` and forwards pool rewards — is never reached. [6](#0-5) 

---

### Likelihood Explanation

- The entry point is fully public; no role, key, or privileged access is required.
- The attacker only needs a valid active staker address (readable from on-chain events) and enough gas to submit one transaction per block.
- The attack is economically rational for a competitor staker who benefits from suppressing rivals' rewards, or for a pure griefing actor.
- On Starknet's sequencer model the attacker submits their transaction first in each block; no mempool gas-price race is needed — submitting before the epoch's first legitimate call suffices.

---

### Recommendation

1. **Remove `disable_rewards` from the public interface** or restrict it to a privileged caller (e.g., `only_security_agent`).
2. **Move the `last_reward_block` write after the `disable_rewards` guard**, so that a no-op call does not consume the block's reward slot.
3. Consider making `last_reward_block` per-staker if the design intent is one reward update per staker per block, rather than one globally.

---

### Proof of Concept

```
Block N:
  Attacker tx:  update_rewards(alice_staker_address, disable_rewards=true)
    → last_reward_block := N
    → returns early, no rewards distributed

  Alice's tx:   update_rewards(alice_staker_address, disable_rewards=false)
    → assert!(N > N)  ← FAILS with REWARDS_ALREADY_UPDATED
    → alice.unclaimed_rewards_own unchanged (zero)

Repeat every block → alice (and all stakers) never accumulate consensus rewards.
``` [7](#0-6)

        fn update_rewards(
            ref self: ContractState, staker_address: ContractAddress, disable_rewards: bool,
        ) {
            self.general_prerequisites();
            let current_block_number = starknet::get_block_number();
            assert!(
                current_block_number > self.last_reward_block.read(),
                "{}",
                Error::REWARDS_ALREADY_UPDATED,
            );

            // Assert staker exists and active.
            // Staker is considered to exist from the moment of `stake` (when `InternalStakerInfo`
            // struct is created) until the calling to `unstake_action` (when `InternalStakerInfo`
            // struct is deleted).
            // Staker remains active until the intent period begins, i.e. K epochs after
            // `unstake_intent` is called.
            let staker_info = self.internal_staker_info(:staker_address);
            let curr_epoch = self.get_current_epoch();
            assert!(
                self.is_staker_active(:staker_address, epoch_id: curr_epoch),
                "{}",
                Error::INVALID_STAKER,
            );

            let staker_pool_info = self.staker_pool_info.entry(staker_address).as_non_mut();
            let (staker_total_strk_balance, staker_total_btc_balance) = self
                .get_staker_total_strk_btc_balance_at_epoch(
                    :staker_address, :staker_pool_info, epoch_id: curr_epoch,
                );
            // Assert staker has non-zero balance.
            // Staker exists with zero balance for the first K epochs after `stake`, then the stake
            // becomes effective.
            assert!(staker_total_strk_balance.is_non_zero(), "{}", Error::INVALID_STAKER);

            // Update last block rewards.
            self.last_reward_block.write(current_block_number);

            if disable_rewards || self.is_pre_consensus() {
                return;
            }

            // Get current block data and update rewards.
            let reward_supplier_dispatcher = self.reward_supplier_dispatcher.read();
            let (strk_block_rewards, btc_block_rewards) = self
                .calculate_block_rewards(:reward_supplier_dispatcher, :curr_epoch);
            self
                ._update_rewards(
                    :staker_address,
                    strk_total_rewards: strk_block_rewards,
                    btc_total_rewards: btc_block_rewards,
                    strk_total_stake: staker_total_strk_balance,
                    btc_total_stake: staker_total_btc_balance,
                    :staker_info,
                    :staker_pool_info,
                    :reward_supplier_dispatcher,
                    :curr_epoch,
                );
        }

        fn _update_rewards(
            ref self: ContractState,
            staker_address: ContractAddress,
            strk_total_rewards: Amount,
            btc_total_rewards: Amount,
            strk_total_stake: NormalizedAmount,
            btc_total_stake: NormalizedAmount,
            mut staker_info: InternalStakerInfoLatest,
            staker_pool_info: StoragePath<InternalStakerPoolInfoV2>,
            reward_supplier_dispatcher: IRewardSupplierDispatcher,
            curr_epoch: Epoch,
        ) {
            // Calculate self rewards.
            let staker_own_rewards = self
                .calculate_staker_own_rewards(
                    :staker_address, :strk_total_rewards, :strk_total_stake, :curr_epoch,
                );

            // Calculate pools rewards.
            let (commission_rewards, total_pools_rewards, pools_rewards_data) = if staker_pool_info
                .has_pool() {
                self
                    .calculate_staker_pools_rewards(
                        :staker_address,
                        :staker_pool_info,
                        :strk_total_rewards,
                        :strk_total_stake,
                        :btc_total_rewards,
                        :btc_total_stake,
                        :curr_epoch,
                    )
            } else {
                (Zero::zero(), Zero::zero(), array![])
            };

            // Update reward supplier.
            let staker_rewards = staker_own_rewards + commission_rewards;
            // Update total rewards.
            reward_supplier_dispatcher
                .update_unclaimed_rewards_from_staking_contract(
                    rewards: staker_rewards + total_pools_rewards,
                );
            // Claim pools rewards.
            claim_from_reward_supplier(
                :reward_supplier_dispatcher,
                amount: total_pools_rewards,
                token_dispatcher: strk_token_dispatcher(),
            );
            // Update staker rewards.
            staker_info.unclaimed_rewards_own += staker_rewards;

            // Update pools rewards.
            let pool_rewards_list = self.update_pool_rewards(:staker_address, :pools_rewards_data);
            // Emit event.
            self
                .emit(
                    Events::StakerRewardsUpdated {
                        staker_address, staker_rewards, pool_rewards: pool_rewards_list.span(),
                    },
                );

            // Write staker rewards to storage.
            self.write_staker_info(:staker_address, :staker_info);
        }
