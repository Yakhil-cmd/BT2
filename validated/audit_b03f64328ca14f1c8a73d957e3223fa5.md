### Title
Staker Can Instantly Set Commission to 100% to Steal All Delegator Rewards — (File: `src/staking/staking.cairo`)

---

### Summary

A staker can call `set_commission` to instantly raise their commission rate to `COMMISSION_DENOMINATOR` (10,000 = 100%) with no timelock, no mandatory notice period, and no enforced cap. This allows a malicious staker to steal all future yield from delegators in their pool at any time.

---

### Finding Description

The `set_commission` function in `src/staking/staking.cairo` permits a staker to set their commission to any value up to `COMMISSION_DENOMINATOR` with immediate, epoch-boundary effect. The only validation is an upper-bound check:

```cairo
assert!(commission <= COMMISSION_DENOMINATOR, "{}", Error::COMMISSION_OUT_OF_RANGE);
``` [1](#0-0) 

The protocol itself acknowledges the gap in a code comment directly above `set_commission_commitment`:

> **Note**: Current commission increase safeguards still allow for sudden commission changes. [2](#0-1) 

The `set_commission_commitment` function exists as a voluntary mechanism for stakers to publicly commit to a maximum commission rate until a given epoch, but it is entirely optional and not enforced. A staker who has never called `set_commission_commitment` — or whose commitment has expired — faces no restriction on instantly raising commission to 100%. [3](#0-2) 

When commission is 100%, the reward distribution logic in the pool allocates the entire epoch reward as staker commission, leaving the pool's cumulative reward index (`Sigma`) unchanged. Delegators who call `claim_rewards` on the pool receive zero for every epoch after the commission change. [4](#0-3) 

---

### Impact Explanation

**High — Theft of unclaimed yield.**

All rewards that delegators would earn in epochs following the commission change are redirected entirely to the staker's reward address. Delegators' principal is not locked (they can exit after the wait window), but every unit of yield accrued after the malicious `set_commission` call is permanently stolen. The attack is silent — no on-chain signal forces delegators to exit before the next reward epoch closes.

---

### Likelihood Explanation

Any registered staker who has opened a delegation pool can execute this at any time with a single transaction. There are no preconditions, no external dependencies, and no privileged access required beyond being a staker. The attack is economically rational: the staker retains their own stake (recoverable after the exit wait window) while capturing 100% of delegator yield indefinitely.

---

### Recommendation

Enforce a mandatory timelock for commission **increases**. When a staker submits a higher commission rate, the new rate should only activate after at least one full epoch (or the `exit_wait_window`), giving delegators observable on-chain notice and time to exit. Commission decreases can remain immediate. This mirrors the intent of `set_commission_commitment` but makes the protection non-optional. [1](#0-0) 

---

### Proof of Concept

1. Staker registers with a 1% commission and calls `set_open_for_delegation` to open a STRK pool.
2. Delegators join the pool over several epochs, accumulating yield tracked in the pool's `cumulative_rewards_trace`.
3. Staker calls `set_commission(10_000)` — `COMMISSION_DENOMINATOR` = 100%.
4. In the next epoch, `update_rewards` distributes epoch rewards; 100% is allocated as staker commission, and the pool's Sigma index does not increase.
5. Every delegator who calls `pool.claim_rewards` receives **0 STRK** for that epoch and all subsequent epochs.
6. Delegators must call `exit_delegation_pool_intent` and wait the full `exit_wait_window` to recover their principal — all yield earned after step 3 is permanently lost to the staker. [5](#0-4) [6](#0-5)

        fn set_commission(ref self: ContractState, commission: Commission) {
            // Prerequisites and asserts.
            self.general_prerequisites();
            assert!(commission <= COMMISSION_DENOMINATOR, "{}", Error::COMMISSION_OUT_OF_RANGE);
            let staker_address = get_caller_address();
            let staker_info = self.internal_staker_info(:staker_address);
            assert!(staker_info.unstake_time.is_none(), "{}", Error::UNSTAKE_IN_PROGRESS);

            let staker_pool_info = self.staker_pool_info.entry(staker_address);
            if let Option::Some(old_commission) = staker_pool_info.commission.read() {
                self
                    .update_commission(
                        :staker_address, :staker_pool_info, :old_commission, :commission,
                    );
            } else {
                staker_pool_info.commission.write(Option::Some(commission));
                self.emit(Events::CommissionInitialized { staker_address, commission });
            }
        }

        fn set_commission_commitment(
            ref self: ContractState, max_commission: Commission, expiration_epoch: Epoch,
        ) {
            self.general_prerequisites();
            assert!(max_commission <= COMMISSION_DENOMINATOR, "{}", Error::COMMISSION_OUT_OF_RANGE);
            let staker_address = get_caller_address();
            let staker_info = self.internal_staker_info(:staker_address);
            assert!(staker_info.unstake_time.is_none(), "{}", Error::UNSTAKE_IN_PROGRESS);
            let staker_pool_info = self.staker_pool_info.entry(staker_address);
            assert!(staker_pool_info.has_pool(), "{}", Error::MISSING_POOL_CONTRACT);
            let current_epoch = self.get_current_epoch();
            if let Option::Some(commission_commitment) = staker_pool_info
                .commission_commitment
                .read() {
                assert!(
                    !self.is_commission_commitment_active(:commission_commitment),
                    "{}",
                    Error::COMMISSION_COMMITMENT_EXISTS,
                );
            }
            // Staker must have a commission since it has a pool.
            let current_commission = staker_pool_info.commission();
            assert!(current_commission <= max_commission, "{}", Error::MAX_COMMISSION_TOO_LOW);
            assert!(expiration_epoch > current_epoch, "{}", Error::EXPIRATION_EPOCH_TOO_EARLY);
            assert!(
                expiration_epoch - current_epoch <= self.get_epoch_info().epochs_in_year(),
                "{}",
                Error::EXPIRATION_EPOCH_TOO_FAR,
            );
            let commission_commitment = CommissionCommitment { max_commission, expiration_epoch };
            staker_pool_info.commission_commitment.write(Option::Some(commission_commitment));
            self
                .emit(
                    Events::CommissionCommitmentSet {
                        staker_address, max_commission, expiration_epoch,
                    },
                );
        }

        fn claim_rewards(ref self: ContractState, pool_member: ContractAddress) -> Amount {
            // Asserts.
            let mut pool_member_info = self.internal_pool_member_info(:pool_member);
            let caller_address = get_caller_address();
            let reward_address = pool_member_info.reward_address;
            assert!(
                caller_address == pool_member || caller_address == reward_address,
                "{}",
                Error::POOL_CLAIM_REWARDS_FROM_UNAUTHORIZED_ADDRESS,
            );

            let until_checkpoint = self.get_current_checkpoint(:pool_member);

            // Calculate rewards and update entry_to_claim_from.
            let (mut rewards, updated_entry_to_claim_from) = self
                .calculate_rewards(
                    :pool_member,
                    from_checkpoint: pool_member_info.reward_checkpoint,
                    :until_checkpoint,
                    entry_to_claim_from: pool_member_info.entry_to_claim_from,
                );
            rewards += pool_member_info._unclaimed_rewards_from_v0;
            pool_member_info._unclaimed_rewards_from_v0 = Zero::zero();
            pool_member_info.entry_to_claim_from = updated_entry_to_claim_from;
            pool_member_info.reward_checkpoint = until_checkpoint;

            // Write the updated pool member info to storage.
            self.write_pool_member_info(:pool_member, :pool_member_info);

            // Transfer rewards to the pool member.
            let reward_token = IERC20Dispatcher { contract_address: STRK_TOKEN_ADDRESS };
            reward_token.checked_transfer(recipient: reward_address, amount: rewards.into());

            // Emit event.
            self
                .emit(
                    Events::PoolMemberRewardClaimed {
                        pool_member, reward_address, amount: rewards,
                    },
                );

            rewards
        }
