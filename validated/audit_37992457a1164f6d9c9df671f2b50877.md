### Title
`MsgStoreBlockList` Admin Authorization Bypass via CheckTx-Only AnteHandler Guard — (`File: app/block_address.go`)

### Summary

`BlockAddressesDecorator.AnteHandle` wraps every authorization check — including the admin-only guard for `MsgStoreBlockList` — inside `if ctx.IsCheckTx()`. In Cosmos SDK v0.50+, the AnteHandler also runs during `FinalizeBlock` with `IsCheckTx() == false`, so the entire guard body is silently skipped at block execution time. A malicious validator can include a crafted `MsgStoreBlockList` transaction directly in a block proposal; honest validators' `ProcessProposal` path (`ValidateTransaction`) checks only whether the *sender is blocked*, not whether the sender is the admin, so the block is accepted. During `FinalizeBlock` the AnteHandler runs again but the admin check is absent, and — absent a redundant server-side check in `msg_server.go` — the message executes, letting an unprivileged address overwrite the on-chain blocklist.

### Finding Description

**Root cause — `app/block_address.go` lines 32–83:**

```go
func (bad BlockAddressesDecorator) AnteHandle(...) (...) {
    if ctx.IsCheckTx() {          // ← entire guard is CheckTx-only
        // ... signer/destination blocked-address checks ...
        admin := bad.getParams(ctx).CronosAdmin
        for _, msg := range tx.GetMsgs() {
            if blocklistMsg, ok := msg.(*types.MsgStoreBlockList); ok {
                if admin != blocklistMsg.From {
                    return ctx, errors.Wrap(sdkerrors.ErrUnauthorized, ...)
                }
            }
        }
    }
    return next(ctx, tx, simulate)   // ← always reached during FinalizeBlock
}
``` [1](#0-0) 

The `if ctx.IsCheckTx()` wrapper means that during `FinalizeBlock` (the state-committing execution mode), `next(ctx, tx, simulate)` is called unconditionally — no signer check, no admin check for `MsgStoreBlockList`.

**Proposal-layer gap — `app/proposal.go` lines 262–335:**

`ProposalHandler.ValidateTransaction` (called during `PrepareProposal`/`ProcessProposal`) checks whether the *sender or destination is on the blocklist*, but contains no check that the sender of a `MsgStoreBlockList` is the admin. [2](#0-1) 

So a block containing a non-admin `MsgStoreBlockList` passes `ProcessProposal` on all honest validators.

**Server-side gap — `x/cronos/keeper/msg_server.go`:**

The admin authorization for `MsgStoreBlockList` is placed exclusively in the AnteHandler. Other privileged messages (`UpdatePermissions`, `UpdateTokenMapping`) carry their own server-side `HasPermission`/admin checks in `msg_server.go`, but the visible code for `StoreBlockList` shows no equivalent guard. [3](#0-2) 

**Attack path:**

1. Attacker (who controls or colludes with a validator) crafts a `MsgStoreBlockList` signed by any non-admin address containing an empty or attacker-chosen blocklist.
2. The validator includes it directly in a block proposal, bypassing their own CheckTx gate.
3. Honest validators run `ProcessProposal` → `ValidateTransaction`: sender is not on the blocklist → accepted.
4. `FinalizeBlock` runs the AnteHandler: `IsCheckTx()` is `false` → admin check is skipped → `next()` is called.
5. `msg_server.StoreBlockList` executes, writing the attacker-supplied blocklist to chain state.
6. `EndBlocker` calls `RefreshBlockList`, loading the new (empty or attacker-controlled) blocklist into the in-memory `ProposalHandler`. [4](#0-3) 

### Impact Explanation

**HIGH** — Bypass of Cronos admin/block-list authorization check with direct security impact.

An attacker can clear or replace the on-chain blocklist, immediately re-enabling sanctioned/blocked addresses to send transactions and receive funds. All subsequent `ValidateTransaction` and `BlockAddressesDecorator` checks use the attacker-supplied list. This is a direct bypass of the governance-controlled security control that the blocklist represents.

### Likelihood Explanation

Requires a malicious or colluding validator to inject the transaction at the block-proposal level (bypassing their own CheckTx). This is a realistic threat model for a chain where validator sets include untrusted parties, and the Cosmos SDK explicitly does not guarantee that validators run CheckTx before including transactions.

### Recommendation

1. **Remove the `if ctx.IsCheckTx()` wrapper** from `BlockAddressesDecorator.AnteHandle` so all checks run during both CheckTx and FinalizeBlock.
2. **Add a server-side admin check** inside `msg_server.StoreBlockList` (mirroring the pattern in `UpdatePermissions`) so authorization is enforced at the message-handler layer regardless of AnteHandler state.
3. **Extend `ProposalHandler.ValidateTransaction`** to reject `MsgStoreBlockList` messages whose sender is not the current `CronosAdmin`, closing the proposal-layer gap.

### Proof of Concept

```
1. Attacker controls validator V.
2. Attacker signs MsgStoreBlockList{From: attacker_addr, Blob: <empty_list>}.
3. V skips CheckTx and includes the tx directly in a PrepareProposal response.
4. Other validators call ProcessProposal → ValidateTransaction:
   - attacker_addr is not in the current blocklist → no rejection.
5. FinalizeBlock runs AnteHandler:
   - ctx.IsCheckTx() == false → entire if-block skipped → next() called.
6. msg_server.StoreBlockList writes empty blob to KVStore.
7. EndBlocker: RefreshBlockList loads empty blob → ProposalHandler.blocklist = {}.
8. All previously blocked addresses are now unblocked on-chain and in-memory.
``` [1](#0-0) [5](#0-4) [4](#0-3)

func (bad BlockAddressesDecorator) AnteHandle(ctx sdk.Context, tx sdk.Tx, simulate bool, next sdk.AnteHandler) (newCtx sdk.Context, err error) {
	if ctx.IsCheckTx() {
		if sigTx, ok := tx.(signing.SigVerifiableTx); ok {
			signers, err := sigTx.GetSigners()
			if err != nil {
				return ctx, err
			}
			for _, signer := range signers {
				if _, ok := bad.blockedMap[sdk.AccAddress(signer).String()]; ok {
					return ctx, fmt.Errorf("signer is blocked: %s", sdk.AccAddress(signer).String())
				}
			}
		}

		for _, msg := range tx.GetMsgs() {
			msgEthTx, ok := msg.(*evmtypes.MsgEthereumTx)
			if ok {
				ethTx := msgEthTx.AsTransaction()
				// check the destination address
				if ethTx.To() != nil {
					if _, ok := bad.blockedMap[sdk.AccAddress(ethTx.To().Bytes()).String()]; ok {
						return ctx, fmt.Errorf("destination address is blocked: %s", sdk.AccAddress(ethTx.To().Bytes()).String())
					}
				}
				// check EIP-7702 authorisation list
				if ethTx.SetCodeAuthorizations() != nil {
					for _, auth := range ethTx.SetCodeAuthorizations() {
						addr, err := auth.Authority()
						if err == nil {
							if _, ok := bad.blockedMap[sdk.AccAddress(addr.Bytes()).String()]; ok {
								return ctx, fmt.Errorf("signer is blocked: %s", addr.String())
							}
						}
						// check the target address
						if _, ok := bad.blockedMap[sdk.AccAddress(auth.Address.Bytes()).String()]; ok {
							return ctx, fmt.Errorf("authorisation address is blocked: %s", sdk.AccAddress(auth.Address.Bytes()).String())
						}
					}
				}
			}
		}

		admin := bad.getParams(ctx).CronosAdmin
		for _, msg := range tx.GetMsgs() {
			if blocklistMsg, ok := msg.(*types.MsgStoreBlockList); ok {
				if admin != blocklistMsg.From {
					return ctx, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
				}
			}
		}
	}
	return next(ctx, tx, simulate)

// SetBlockList don't fail if the identity is not set or the block list is empty.
func (h *ProposalHandler) SetBlockList(blob []byte) error {
	if h.Identity == nil {
		return nil
	}

	if bytes.Equal(h.lastBlockList, blob) {
		return nil
	}
	h.lastBlockList = make([]byte, len(blob))
	copy(h.lastBlockList, blob)

	if len(blob) == 0 {
		h.blocklist = make(map[string]struct{})
		return nil
	}

	reader, err := age.Decrypt(bytes.NewBuffer(blob), h.Identity)
	if err != nil {
		return err
	}

	data, err := io.ReadAll(reader)
	if err != nil {
		return err
	}

	var blocklist BlockList
	if err := json.Unmarshal(data, &blocklist); err != nil {
		return err
	}

	// convert to map
	m := make(map[string]struct{}, len(blocklist.Addresses))
	for _, s := range blocklist.Addresses {
		addr, err := h.addressCodec.StringToBytes(s)
		if err != nil {
			return fmt.Errorf("invalid bech32 address: %s, err: %w", s, err)
		}
		if IsUnblockable(addr) {
			continue
		}
		encoded, err := h.addressCodec.BytesToString(addr)
		if err != nil {
			return fmt.Errorf("invalid bech32 address: %s, err: %w", s, err)
		}
		m[encoded] = struct{}{}
	}

	h.blocklist = m
	return nil
}

func (h *ProposalHandler) ValidateTransaction(tx sdk.Tx, txBz []byte) error {
	if len(h.blocklist) == 0 {
		// fast path, accept all txs
		return nil
	}

	var err error
	if tx == nil {
		tx, err = h.TxDecoder(txBz)
		if err != nil {
			return err
		}
	}

	sigTx, ok := tx.(signing.SigVerifiableTx)
	if !ok {
		return fmt.Errorf("tx of type %T does not implement SigVerifiableTx", tx)
	}

	signers, err := sigTx.GetSigners()
	if err != nil {
		return err
	}
	for _, signer := range signers {
		encoded, err := h.addressCodec.BytesToString(signer)
		if err != nil {
			return fmt.Errorf("invalid bech32 address: %s, err: %w", signer, err)
		}
		if _, ok := h.blocklist[encoded]; ok {
			return fmt.Errorf("signer is blocked: %s", encoded)
		}
	}

	for _, msg := range tx.GetMsgs() {
		msgEthTx, ok := msg.(*evmtypes.MsgEthereumTx)
		if ok {
			ethTx := msgEthTx.AsTransaction()
			// check the destination address
			if ethTx.To() != nil {
				encoded, err := h.addressCodec.BytesToString(ethTx.To().Bytes())
				if err != nil {
					return fmt.Errorf("invalid bech32 address: %s, err: %w", ethTx.To(), err)
				}
				if _, ok := h.blocklist[encoded]; ok {
					return fmt.Errorf("destination address is blocked: %s", encoded)
				}
			}
			// check EIP-7702 authorisation list
			if ethTx.SetCodeAuthorizations() != nil {
				for _, auth := range ethTx.SetCodeAuthorizations() {
					addr, err := auth.Authority()
					if err == nil {
						encoded, err := h.addressCodec.BytesToString(addr.Bytes())
						if err != nil {
							return fmt.Errorf("invalid bech32 address: %s, err: %w", addr, err)
						}
						if _, ok := h.blocklist[encoded]; ok {
							return fmt.Errorf("signer is blocked: %s", encoded)
						}
					}
					// check the target address
					encoded, err := h.addressCodec.BytesToString(auth.Address.Bytes())
					if err != nil {
						return fmt.Errorf("invalid bech32 address: %s, err: %w", auth.Address, err)
					}
					if _, ok := h.blocklist[encoded]; ok {
						return fmt.Errorf("authorisation address is blocked: %s", encoded)
					}
				}
			}
		}
	}

	return nil

func (k msgServer) UpdatePermissions(goCtx context.Context, msg *types.MsgUpdatePermissions) (*types.MsgUpdatePermissionsResponse, error) {
	ctx := sdk.UnwrapSDKContext(goCtx)
	admin := k.Keeper.GetParams(ctx).CronosAdmin
	// if admin is empty, no sender could be equal to it
	if admin != msg.From {
		return nil, errors.Wrap(sdkerrors.ErrUnauthorized, "msg sender is not authorized")
	}
	acc, err := sdk.AccAddressFromBech32(msg.Address)
	if err != nil {
		return nil, err
	}
	k.SetPermissions(ctx, acc, msg.Permissions)

	return &types.MsgUpdatePermissionsResponse{}, nil

func (app *App) EndBlocker(ctx sdk.Context) (sdk.EndBlock, error) {
	rsp, err := app.ModuleManager.EndBlock(ctx)
	if err := app.RefreshBlockList(ctx); err != nil {
		app.Logger().Error("failed to update blocklist", "error", err)
	}
	return rsp, err
}

func (app *App) RefreshBlockList(ctx sdk.Context) error {
	// refresh blocklist
	return app.blockProposalHandler.SetBlockList(app.CronosKeeper.GetBlockList(ctx))
